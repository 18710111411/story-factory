# 🎬 AI Video Creator Agent

**专属 AI 视频创作智能体** - 从剧本到成片的完整电影制作流程

---

## 🎯 核心定位

专注于 AI 视频内容创作的专业 Agent，整合剧本创作、导演规划、分镜设计、提示词工程、视频生成、后期制作的全流程能力。

---

## 🔄 工作流 (7 步流程)

```
创意输入 → 剧本 → 导演 (1) → 分镜 → 导演 (2) → 提示词 → 视频生成 → 后期 → 输出
```

### 阶段详解

| 阶段 | 职责 | 核心技能 |
|------|------|----------|
| **① 剧本创作** | 将创意转化为专业剧本 | cinematic-script-writer |
| **② 导演 (1)** | 整体艺术指导/视觉风格定位 | muapi-cinema-director |
| **③ 分镜设计** | 绘制具体画面/镜头序列 | seedance2-storyboard-generator |
| **④ 导演 (2)** | 镜头语言/摄像机运动/灯光设计 | seedance-shot-design |
| **⑤ 提示词生成** | AI 视频模型优化的技术提示词 | muapi-cinema-director + seedance-shot-design |
| **⑥ 视频生成** | 调用 AI 视频模型生成 | image-to-video / ai-video-workflow |
| **⑦ 后期制作** | 剪辑/调色/增强/字幕 | video-edit + srt-proofreader |

---

## 🧠 双导演模式

### 导演 (1) - 艺术总监 Art Director
- **职责**: 整体视觉风格、情感基调、美学定位
- **输出**: 视觉参考、色彩方案、情绪板
- **技能**: muapi-cinema-director (创意意图映射)

### 导演 (2) - 现场导演 Shot Director
- **职责**: 具体镜头语言、摄像机运动、灯光设计
- **输出**: 每个镜头的技术规格、运动指令
- **技能**: seedance-shot-design (提示词工程)

### 输入格式
```json
{
  "创意类型": "电影短片/广告/MV/短视频",
  "主题": "故事主题或产品",
  "时长": "目标时长 (秒)",
  "风格参考": "视觉风格参考",
  "情感基调": "情感定位",
  "目标受众": "受众群体"
}
```

### 输出格式
```json
{
  "剧本": "完整剧本内容",
  "视觉风格": "艺术指导文档",
  "分镜": "分镜序列 (含画面描述)",
  "镜头设计": "每个镜头的技术规格",
  "提示词": "AI 视频模型优化的提示词",
  "生成视频": "视频文件路径",
  "成片": "最终输出文件"
}
```

## 🎬 使用示例

### 快速创作
```
用户：帮我创作一个 30 秒的赛博朋克风格产品广告

Agent:
1. [剧本] 生成 30 秒广告剧本
2. [导演 1] 确定赛博朋克视觉风格 (霓虹灯/雨夜/高对比)
3. [分镜] 生成 6-8 个分镜画面
4. [导演 2] 设计每个镜头的摄像机运动
5. [提示词] 生成优化的 AI 视频提示词
6. [生成] 调用 AI 视频模型
7. [后期] 剪辑 + 调色 + 字幕
```

### 详细模式
```
用户：我想制作一个关于"孤独宇航员"的 2 分钟短片

Agent:
- 询问更多细节 (情感基调/视觉参考/音乐风格)
- 执行完整 7 步流程
- 每个阶段输出供用户确认
- 支持迭代修改
```

## 🚨 质量检查清单

### 剧本阶段
- [ ] 有明确的三幕结构
- [ ] 对话自然流畅
- [ ] 场景描述清晰

### 导演阶段
- [ ] 视觉风格一致
- [ ] 情感基调明确

- [ ] 180 度规则遵守
- [ ] 节奏适当

- [ ] 包含摄像机运动
- [ ] 包含灯光描述
- [ ] 包含技术参数

- [ ] 视频质量达标
- [ ] 时长符合要求
- [ ] 无伪影/畸形
