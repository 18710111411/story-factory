# 🎬 AI Video Creator - 快速开始指南

## 🚀 5 分钟快速创作

### 第一步：准备创意简报

```markdown
主题：{一句话描述故事/产品}
时长：{15 秒/30 秒/60 秒/2 分钟}
风格：{赛博朋克/浪漫/悬疑/喜剧/...}
情感：{激昂/温馨/紧张/幽默/...}
参考：{参考电影/视频/图片链接}
```

### 第二步：启动工作流

```
用户：帮我创作一个 30 秒的赛博朋克风格产品广告
产品：智能手表
核心卖点：全天候健康监测

Agent 将自动执行 7 步流程...
```

### 第三步：确认关键节点

工作流会在以下节点等待确认：
- ✅ 剧本完成后
- ✅ 视觉风格确定后
- ✅ 分镜生成后
- ✅ 提示词生成后

### 第四步：生成与导出

- AI 视频生成 (异步，需等待)
- 后期处理
- 导出最终成片

## 📋 完整使用流程

### 模式 A：快速创作 (推荐新手)

```
用户：创作一个{时长}的{风格}{主题}视频

Agent 自动执行：
1. 生成剧本 (自动)
2. 确定视觉风格 (自动)
3. 生成分镜 (自动)
4. 设计镜头 (自动)
5. 生成提示词 (自动)
6. 调用 AI 生成 (需确认)
7. 后期处理 (自动)

输出：完整视频
```

### 模式 B：协作创作 (推荐专业用户)

```
用户：我想制作一个{主题}视频，我们一起创作

Agent 分步引导：
1. [剧本] 生成初稿 → 用户修改 → 确认
2. [导演 1] 提出视觉方案 → 用户选择 → 确认
3. [分镜] 生成分镜 → 用户调整 → 确认
4. [导演 2] 设计镜头 → 用户审阅 → 确认
5. [提示词] 生成提示词 → 用户审核 → 确认
6. [生成] 开始生成 → 用户确认
7. [后期] 处理完成 → 用户验收

输出：完整视频 + 所有中间文件
```

### 模式 C：单阶段使用

```
用户：请帮我{具体任务}

例如：
- "帮我把这个剧本转成分镜"
- "为这个场景设计镜头"
- "优化这个 AI 视频提示词"
- "评估这个视频的质量"

Agent 执行指定阶段任务
```

## ⚙️ 配置指南

### API Key 配置

创建配置文件 `~/.openclaw/workspace/agents/ai-video-creator/config.json`:

```json
{
  "muapi": {
    "api_key": "your_muapi_key",
    "default_model": "kling"
  },
  "seedance": {
    "api_key": "your_jimeng_key",
    "version": "2.0"
  },
  "video_generation": {
    "provider": "image-to-video",
    "api_key": "your_api_key"
  },
  "preferences": {
    "default_duration": 30,
    "default_resolution": "1920x1080",
    "default_fps": 24,
    "auto_post_production": true
  }
}
```

### 模型选择建议

| 场景 | 推荐模型 | 原因 |
|------|----------|------|
| 高质量美学镜头 | Veo3 | 最佳画质/光影 |
| 复杂角色动作 | Kling | 动作自然/物理准确 |
| 快节奏场景 | Luma | 生成快/能量感强 |
| 产品广告 | Kling | 产品展示清晰 |
| 风景/环境 | Veo3 | 环境细节丰富 |

## 🔧 故障排查

### 问题：视频生成失败

**可能原因**:
- API Key 无效
- 提示词过长/过短
- 模型过载

**解决方案**:
1. 检查 API Key 配置
2. 简化提示词
3. 更换模型重试

### 问题：分镜与剧本不符

**解决方案**:
1. 返回剧本阶段修改
2. 在分镜提示词中强调关键场景
3. 手动调整分镜描述

### 问题：视频质量不佳

**解决方案**:
1. 优化提示词 (添加更多细节)
2. 更换更合适的模型
3. 调整技术参数 (分辨率/时长)
4. 使用 video-edit 进行后期增强

## 🎓 进阶技巧

### 提示词优化技巧

1. **具体胜于抽象**
   - ❌ "美丽的风景"
   - ✅ "日落时分的雪山，金色阳光洒在山峰，前景是高山草甸"

2. **使用电影术语**
   - "Dolly In 营造亲密感"
   - "Dutch Angle 表现不安"
   - "Chiaroscuro 灯光增加戏剧性"

3. **添加物理逻辑**
   - "雨水在霓虹灯下反射出彩色光斑"
   - "风吹动头发，衣服随风飘动"

4. **指定技术参数**
   - "24fps, 2.39:1, shallow DOF"
   - "anamorphic lens flares"

### 迭代策略

1. **小步快跑**: 先测试 5 秒片段，确认风格后再完整生成
2. **A/B 测试**: 同一场景生成 2-3 个版本，选择最佳
3. **分层生成**: 复杂场景分解为多个镜头分别生成
4. **后期补救**: 小问题用 video-edit 修复，避免重新生成
