## 🎯 核心原则

### 规则 1: 镜头内变化优先于切镜头

**原则**: 能用一个镜头的内部变化讲清楚，就不要切镜头。

**理由**:
- ✅ 视觉更流畅，情绪不中断
- ✅ 减少渲染次数，降低成本
- ✅ CG 动画优势最大化
- ✅ 对话完整性更好保持

**镜头内变化技巧**:
| 技巧 | 说明 | 示例 |
|------|------|------|
| **Pull Out/Push In** | 连续拉远/推进 | 眼睛特写→全景 |
| **Rack Focus** | 焦点转换 | 门缝→角色 |
| **Tracking** | 跟随拍摄 | 角色行走 |
| **180 度环绕** | 围绕角色旋转 | 气场强调 |
| **连续表演** | 情绪弧线不切 | 委屈→愤怒 |
| **动作连续** | 物理反应链 | 拍门→震动→灰尘 |
| **特效 + 表演** | 同时呈现 | 系统激活 + 反应 |

**目标镜头密度**:
- 真人短剧：15-20 镜头/分钟
- **CG 动画：18-22 镜头/分钟** (非 30+)
- 电影质感：10-15 镜头/分钟

---

### 规则 2: 对话完整保留优先

**原则**: 宁可调整镜头节奏，不删减核心对话。

**对话处理策略**:
| 方法 | 节省时间 | 适用场景 |
|------|----------|----------|
| 语速 1.3 倍 | -30% | 大部分对话 |
| 部分重叠 | -10% | 非关键台词 |
| 画外音处理 | -5% | 情绪性台词 |
| 机械音加速 | -20% | 系统 VO |

**对话时长计算**:
```
正常语速：180 字/分钟
1.3 倍速：234 字/分钟
机械音：240 字/分钟 (可接受)
```

**口型同步要求**:
- 精确同步 (CG 优势)
- 情绪驱动口型 (非机械)
- 呼吸细节保留

---

### 规则 3: CG 动画优势最大化

**CG 相比真人的优势**:
| 优势 | 应用 |
|------|------|
| 摄像机自由 | 不受轨道/摇臂限制 |
| 焦点完美控制 | 伺服对焦，平滑转换 |
| 表演可重复 | 无需重拍，关键帧调整 |
| 时间可控 | 慢动作/快进无成本 |
| 特效原生 | 粒子/全息/光芒内置 |
| 物理模拟 | 头发/衣服/灰尘自动 |

**必须利用的 CG 优势**:
1. ✅ 连续镜头内变化 (非切镜头)
2. ✅ 完美焦点转换 (Rack Focus)
3. ✅ 复杂特效 (系统/粒子/光芒)
4. ✅ 物理模拟 (头发/衣服/灰尘)
5. ✅ 慢动作 (关键瞬间)

---

### 规则 4: 情绪弧线连续

**原则**: 角色的情绪变化应该连续呈现，不要切断。

**情绪弧线示例**:
```
麻木 → 委屈 → 质问 → 愤怒 → 决绝
  ↓       ↓       ↓       ↓       ↓
连续表演，不切镜头
```

**单镜头情绪变化技巧**:
- 眉毛：平直 → 微皱 → 紧皱
- 眼睛：空洞 → 含泪 → 坚定
- 嘴巴：闭合 → 微张 → 咬牙
- 下巴：低垂 → 平视 → 微抬

**镜头时长建议**:
- 简单情绪：2-3 秒
- **情绪变化：5-7 秒** (给足时间)
- 高潮情绪：3-5 秒 (慢动作可延长)

---

### 规则 5: 16:9 横屏构图优化

**横屏 vs 竖屏**:
| 场景 | 竖屏 9:16 | 横屏 16:9 |
|------|-----------|-----------|
| 特写 | 脸部占满 | 脸部 + 环境 |
| 对话 | 单人为主 | 双人/多人 |
| 空间 | 压抑/狭窄 | 开阔/沉浸 |
| 信息 | 聚焦主体 | 主体 + 背景 |

**横屏构图技巧**:
- 三分法：主体置左/右 1/3 处
- 引导线：利用环境线条
- 景深：背景虚化突出主体
- 负空间：留白创造氛围

## 📐 分镜设计规范

### AI 视频模型时长限制

| 模型 | 最长时长 | 推荐时长 | 备注 |
|------|----------|----------|------|
| **Seedance (即梦)** | **15 秒** | 10-15 秒 | 超过需分段拼接 |
| Kling | 10 秒 | 5-10 秒 | 可延长至 30 秒 |
| Veo3 | 60 秒 | 30-60 秒 | 高质量长镜头 |
| Luma | 10 秒 | 5-10 秒 | 快节奏优化 |

**Seedance 15 秒规则**:
- 单个镜头 ≤ 15 秒
- 超过 15 秒需拆分为多个镜头
- 拆分点选择：动作/焦点/对话自然停顿处
- 拼接方式：后期剪辑或连续生成

### 对话时长计算规则

**核心公式**:
```
镜头时长 (秒) = 台词字数 / (语速 字/分钟 / 60)

正常语速：180 字/分钟 = 3 字/秒
1.3 倍速：234 字/分钟 = 3.9 字/秒
1.5 倍速：270 字/分钟 = 4.5 字/秒
机械音：300 字/分钟 = 5 字/秒
```

**镜头时长与对话匹配**:
| 镜头时长 | 最大字数 (正常) | 最大字数 (1.3 倍) | 适用场景 |
|----------|-----------------|-------------------|----------|
| 3 秒 | 9 字 | 12 字 | 短句/反应 |
| 4 秒 | 12 字 | 16 字 | 简单对话 |
| 5 秒 | 15 字 | 20 字 | 标准对话 |
| 6 秒 | 18 字 | 23 字 | 长对话 |
| 8 秒 | 24 字 | 31 字 | 复杂对话 |
| 10 秒 | 30 字 | 39 字 | 独白/VO |

**对话分配优先级**:
1. ✅ 关键台词 (完整保留)
2. ✅ 情绪台词 (可精简)
3. ⚠️ 解释台词 (大幅精简/画外音)
4. ⚠️ 冗余台词 (删除)

**转场方式优先级**:
1. ✅ 连续镜头内转换 (最优)

3. ✅ 特效转场 (粒子消散)
4. ✅ 黑场 (0.5-1s)
5. ⚠️ 硬切 (最后选择)

## 🎯 质量检查清单

### 对话 - 镜头时长检查
```
□ 每个镜头的对话字数在时长限制内
□ 语速合理 (1.0-1.5 倍，机械音可达 1.7 倍)
□ 关键台词有足够时长 (非匆忙带过)
□ 情绪台词有停顿空间 (非填满)
□ 画外音/重叠处理合理
□ 总对话时长 ≤ 总时长的 70%
```

### 前期检查
```
□ 镜头密度合理 (18-22 镜头/分钟)
□ 对话完整保留 (语速/重叠优化)
□ 镜头内变化最大化
□ CG 优势充分利用
□ 情绪弧线连续
```

### 镜头检查
```
□ 每个镜头有内部变化 (非静态)
□ 焦点/运动/表演至少一种变化
□ 时长与情绪匹配
□ 灯光符合情绪
□ 构图符合 16:9 优化
```

### 对话检查
```
□ 全部台词保留
□ 语速 1.3 倍 (大部分)
□ 口型精确同步
□ 情绪驱动表演
□ 呼吸细节保留
```

### 特效检查
```
□ 粒子效果细腻
□ 全息界面半透明
□ 光芒体积感
□ 物理模拟准确
□ 与表演同步
```

## 🎊 总结

**导演核心思想**:
> "用镜头内变化代替切镜头，用连续表演代替碎片剪辑，用 CG 优势代替真人限制。"

**目标**:
- 视觉流畅 (非碎片化)
- 对话完整 (非删减)
- 成本可控 (非过度渲染)
- 情绪连贯 (非中断)
