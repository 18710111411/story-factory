# 📝 番茄小说投稿指南

**平台**: 番茄免费小说  
**审核时间**: 1-3 个工作日  
**签约方式**: 自动签约 + 人工签约

---

## 📋 投稿前准备

### 1. 书名优化

**要求**:
- 长度：10-20 字
- 包含核心梗
- 有爽点/悬念

**示例**:
| 类型 | 书名示例 |
|------|---------|
| 玄幻 | 《重生后，我靠社死变强》 |
| 都市 | 《离婚后，前妻跪求复合》 |
| 言情 | 《替嫁后，傅少宠我上天》 |
| 悬疑 | 《我在地府当公务员》 |

### 2. 简介优化

**结构**:
```
【第 1 行】核心梗（抓住读者）
【第 2-3 行】金手指展示
【第 4-5 行】核心冲突
【第 6 行】悬念钩子
```

**示例**:
```
300 年后的合欢老祖许溪，重生回到 300 年前。
获得"社死系统"——只要让人尴尬就能变强！

开局被捉奸在床，被迫负责合欢宗师姐；
外门大比跳舞夺冠，震惊整个青云宗；
正魔大战布局天下，揭开天道阴谋……

这一世，他要改变所有遗憾！
```

### 3. 黄金三章

**第 1 章要求**:
- [ ] 主角登场
- [ ] 金手指觉醒
- [ ] 核心冲突出现
- [ ] 章末钩子

**第 2 章要求**:
- [ ] 展示金手指能力
- [ ] 第一个小爽点
- [ ] 引入重要配角

**第 3 章要求**:
- [ ] 首次打脸/冲突
- [ ] 读者期待感建立
- [ ] 引出后续剧情

## 📋 投稿流程

### 步骤 1: 注册账号

1. 下载番茄作家助手 APP
2. 手机号注册
3. 实名认证（需要身份证）

### 步骤 2: 创建作品

1. 点击"创建新书"
2. 填写书名、类型、简介
3. 上传封面（可 AI 生成）

### 步骤 3: 发布章节

1. 上传前 3 章（至少 6000 字）
2. 点击"申请签约"
3. 等待审核（1-3 个工作日）

### 步骤 4: 签约后

1. 自动获得推荐流量
2. 保持日更（建议日更 4000+）
3. 达到 10 万字可申请"爆款推荐"

## 📊 番茄平台规则

### 签约标准

| 标准 | 要求 |
|------|------|
| 字数 | 前 3 章 6000+ 字 |
| 质量 | 无抄袭、无违规 |
| 更新 | 签约后日更 4000+ |

### 推荐机制

| 阶段 | 字数 | 推荐量 |
|------|------|--------|
| 新书期 | 0-10 万字 | 1000-5000 阅读 |
| 成长期 | 10-30 万字 | 5000-50000 阅读 |
| 爆发期 | 30-100 万字 | 5 万 + 阅读 |
| 爆款 | 100 万 + 字 | 10 万 + 阅读 |

### 收益计算

```
收益 = 广告分成 + 付费阅读 + 全勤奖 + 打赏

广告分成：阅读时长 × 单价（约 0.01-0.05 元/分钟）
付费阅读：读者付费章节 × 分成比例（50%）
全勤奖：日更 4000+，每月 600-1500 元
打赏：读者打赏 × 分成比例（50%）
```

## 📈 爆款公式

### 番茄爆款特点

1. **节奏快** - 3 章内必须有爽点
2. **金手指强** - 系统/重生/穿越
3. **情绪爽** - 打脸/装逼/收获
4. **更新稳** - 日更 4000+ 不断更
