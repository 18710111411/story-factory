## 📋 人设生成器

### 快速生成公式

```
姓名 + 年龄 + 身份 + 性格关键词 + 核心动机 + 秘密/伏笔

示例：
许溪，16 岁，青云宗外门弟子，
性格：老谋深算 + 谨慎多疑 + 乐子人，
核心动机：在宗门覆灭前逃跑，
秘密：重生者，前世是合欢宗老祖。
```

## 📋 人设检查清单

创建角色时检查：

- [ ] 姓名是否有辨识度
- [ ] 年龄是否符合剧情
- [ ] 身份是否合理
- [ ] 性格是否有特点（2-3 个关键词）
- [ ] 核心动机是否明确
- [ ] 是否有秘密/伏笔
- [ ] 是否会成长/变化
- [ ] 是否有遗憾/缺点
