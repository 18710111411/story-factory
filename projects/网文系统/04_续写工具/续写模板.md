# ✍️ 续写工具模板

**用途**: 日更 10 万字核心工具  
**效率**: 2 小时/5 万字

---

## 📋 续写流程

### 步骤 1: 输入前情提要

```markdown
【前情提要】
- 上一章结尾：许溪击败对手，获得神秘玉简
- 当前状态：炼气三层，积分 140
-  pending 任务：调查陈长生
- 情绪基调：警惕 + 疑惑
```

### 步骤 2: 输入本章大纲

```markdown
【本章大纲】
- 核心事件：许溪返回住处，研究玉简
- 爽点：玉简内有惊喜
- 字数：10000 字
- 关键对话：许溪 vs 系统
```

### 步骤 3: 生成续写

**提示词**:
```
根据以下信息续写小说章节：

【小说信息】
书名：《穿越后：我靠社死升级》
类型：玄幻/重生/系统
文风：轻松 + 爽文

【前情提要】
（粘贴步骤 1 内容）

【本章大纲】
（粘贴步骤 2 内容）

【要求】
1. 字数：10000 字
2. 对话比例：30-40%
3. 爽点密度：每千字 1 个爽点
4. 章末钩子：必须有
5. 避免：重复描写、无关环境描写
```

他从怀中取出那枚神秘的玉简，玉简温润，散发着淡淡的灵气波动。

"这东西，到底是什么？"

许溪神识探入玉简，下一秒，大量信息涌入脑海。

......
```

### 战斗场景扩写

**输入**:
```
许溪与王猛对战，轻松获胜。
```

**输出**:
```
"许溪，受死吧！"

王猛大喝一声，手中长剑化作一道白光，直刺许溪咽喉。

台下观众纷纷屏住呼吸，这一剑太快了！

然而，许溪却站在原地，动都未动。

就在剑尖即将触及他喉咙的瞬间——

他动了。

只是简单地侧身，便躲过了这致命一击。

然后，抬手，一指点出。

"砰！"

王猛整个人倒飞出去，重重摔在地上，昏死过去。

全场寂静。

......
```

### 情感场景扩写

**输入**:
```
苏清雪向许溪表白，许溪拒绝。
```

**输出**:
```
"许师兄，我……我喜欢你。"

苏清雪低着头，声音细若蚊呐，脸颊红得像熟透的苹果。

许溪看着她，心中一软。

但他知道，自己不能接受。

"清雪，你听我说。"许溪轻声道，"我……不是一个值得你喜欢的人。"

"为什么？"苏清雪抬起头，眼中含着泪光。

"因为……"许溪沉默片刻，"因为我迟早要离开这里。"

## 📋 润色模板

### 润色指令

```
请润色以下段落，要求：

1. 保持原意不变
2. 增加五感描写（视觉/听觉/嗅觉/触觉/味觉）
3. 优化对话节奏
4. 删除冗余词汇
5. 增强画面感

【待润色段落】
（粘贴内容）
```

### 润色示例

**原文**:
```
许溪很生气，看着陈长生，问他为什么要骗自己。
```

**润色后**:
```
许溪握紧了拳头，指甲嵌入掌心，渗出丝丝血迹。

他盯着陈长生，眼神冰冷如刀。

"为什么？"

三个字，仿佛从牙缝中挤出。

陈长生避开他的目光，沉默不语。
```

## ⚠️ 注意事项

1. **不要一次性生成太多** - AI 容易重复，建议分章节生成
2. **每章检查一致性** - 人物性格、修为、时间线
3. **敏感词检测** - 番茄审核严格，必须检测
4. **保留修改空间** - AI 生成的内容可以修改
