## 简介生成

### 简介公式

```
【第 1 行】核心梗（抓住读者）
【第 2-3 行】金手指展示
【第 4-5 行】核心冲突
【第 6 行】悬念钩子
```

### 玄幻简介示例

```
300 年后的合欢老祖许溪，重生回到 300 年前。
获得"社死系统"——只要让人尴尬就能变强！

开局被捉奸在床，被迫负责合欢宗师姐；
外门大比跳舞夺冠，震惊整个青云宗；
正魔大战布局天下，揭开天道阴谋……

这一世，他要改变所有遗憾！
```

### 都市简介示例

```
重生回到 2010 年，林默发现自己获得了未来记忆。
买彩票、炒房、投资互联网……

前世看不起他的人，这一世都要仰望他。
但当他站在巅峰时，却发现……

这一世，真的能改变一切吗？
```

### 言情简介示例

```
被迫替嫁，她以为会嫁给一个老男人。
没想到，新郎竟是京城最神秘的总裁。

"签了这份协议，做我的契约妻子。"
"一年后，还你自由。"

她以为只是一场交易，
却没想到，总裁当真了……
```

- 《重生 2000 年》

### 技巧 2: 反差对比

- 《从保安到首富》

- 《离婚后，前夫跪求复合》
- 《真相曝光，所有人震惊了》

### 技巧 4: 情绪共鸣

- 《这一世，不再当舔狗》
- 《莫欺少年穷》
- 《这一次，我要赢》

## 简介检查清单

- [ ] 前 3 行抓住读者
- [ ] 突出金手指
- [ ] 突出核心冲突
- [ ] 留悬念钩子
- [ ] 字数 100-300 字
- [ ] 符合平台调性

---

## 平台适配

### 番茄/七猫

- 书名直白，包含核心梗
- 简介突出爽点
- 节奏快，不绕弯子

### 晋江

- 书名文艺一些
- 简介突出感情线
- 文笔细腻

### 起点

- 书大气一些
- 简介突出世界观
- 设定新颖
