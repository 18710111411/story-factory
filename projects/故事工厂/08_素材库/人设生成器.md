## 人设生成公式

### 快速生成公式

```
姓名 + 年龄 + 身份 + 性格关键词 + 核心动机 + 秘密/伏笔

示例：
许溪，16 岁，青云宗外门弟子，
性格：老谋深算 + 谨慎多疑 + 乐子人，
核心动机：在宗门覆灭前逃跑，
秘密：重生者，前世是合欢宗老祖。
```

## 人设检查清单

创建角色时检查：

- [ ] 姓名是否有辨识度
- [ ] 年龄是否符合剧情
- [ ] 身份是否合理
- [ ] 性格是否有特点（2-3 个关键词）
- [ ] 核心动机是否明确
- [ ] 是否有秘密/伏笔
- [ ] 是否会成长/变化
- [ ] 是否有遗憾/缺点

## 角色成长弧线

### 设计方法

```
初始状态 → 经历事件 1 → 经历事件 2 → 最终状态

示例：
懦弱少年 → 被退婚受辱 → 获得金手指 → 成为强者
```

### 成长类型

| 类型 | 说明 |
|------|------|
| 实力成长 | 修为/能力提升 |
| 心智成长 | 性格/思想成熟 |
| 地位成长 | 身份/地位提升 |
| 感情成长 | 感情观成熟 |
