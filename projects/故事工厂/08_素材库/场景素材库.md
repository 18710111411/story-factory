## 战斗场景

### 低境界战斗（炼气/筑基）

```
【开场】
两人对峙，气势碰撞，周围空气仿佛凝固。

【过程】
- 功法对撞：X 色光芒交织，灵力激荡
- 身法闪躲：残影连连，险之又险
- 法宝碰撞：铿锵之声，火星四溅

【收尾】
一方倒地，胜负已分。胜者负手而立，衣袂飘飘。
```

### 高境界战斗（金丹/元婴）

```
【开场】
威压笼罩，方圆百里灵力沸腾，天地变色。

【过程】
- 领域展开：X 色领域覆盖，规则之力涌动
- 神通对轰：山崩地裂，空间扭曲
- 底牌尽出：秘法/禁术/燃烧精血

【收尾】
天地恢复平静，只剩废墟。胜者气息萎靡，但眼神坚定。
```

