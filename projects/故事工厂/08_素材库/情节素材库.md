## 开篇套路

### 套路 1: 退婚开局

```
1. 未婚妻上门退婚
2. 主角被嘲讽
3. 金手指觉醒
4. 立下三年之约
5. "莫欺少年穷"
```

### 套路 2: 重生开局

```
1. 主角从梦中醒来
2. 发现重生回到十年前
3. 利用未来知识获利
4. 弥补前世遗憾
5. 开始布局
```

### 套路 3: 系统开局

```
1. 主角处于困境
2. 被嘲讽/被欺负
3. 系统突然觉醒
4. 获得第一个任务
5. 完成任务变强
```

### 套路 4: 穿越开局

```
1. 在陌生环境醒来
2. 原主有困境
3. 利用现代知识破局
4. 引起注意
5. 开始崛起
```

## 爽点套路

### 打脸型

```
【铺垫】
反派嘲讽："你这种废物，也配？"

【蓄势】
主角不怒反笑："是吗？"

【爆发】
一招制敌，反派倒地。

【震惊】
围观群众："这……这怎么可能？！"
```

### 收获型

```
【探索】
主角进入秘境/遗迹

【发现】
发现宝物/功法/传承

【获得】
成功获得，实力提升

【震惊】
"这可是 XXX 级别的宝物！"
```

### 成长型

```
【瓶颈】
主角卡在瓶颈很久

【突破】
闭关修炼，终于突破

【展示】
出关后，实力大涨

【震惊】
"你……你突破了？！"
```

### 守护型

```
【危机】
重要的人被威胁

【爆发】
主角暴怒："动她者，死！"

【战斗】
爆发全部实力

【胜利】
成功守护，感情升温
```

## 伏笔套路

```

【推进】

【揭秘】
玉佩是身份信物，主角是 XXX 后人
```

```
【埋设】

【推进】

【揭秘】
```
