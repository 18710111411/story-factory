# 文章润色工具

**位置**: `03_检查清单/之前`  
**用途**: 初稿完成后、自检之前使用

---

## 润色模式

### 模式 1: 精修模式（重要章节）

**适用**: 大高潮、关键转折、感情戏

**润色重点**:
- [ ] 五感描写增强
- [ ] 修辞手法优化
- [ ] 情绪渲染加深
- [ ] 节奏控制（张弛有度）
- [ ] 对话精炼

**操作**:
1. 复制初稿到润色工具
2. 选择"精修模式"
3. 逐段润色
4. 对比原文，保留精华

---

### 模式 2: 快速模式（日常章节）

**适用**: 过渡章、日常章

**润色重点**:
- [ ] 错别字/标点修正
- [ ] 重复用词替换
- [ ] 明显 AI 味消除
- [ ] 段落划分优化

**操作**:
1. 复制初稿到润色工具
2. 选择"快速模式"
3. 自动 + 手动结合
4. 5-10 分钟完成

---

### 模式 3: 去 AI 味模式（AI 生成后必用）

**适用**: 所有 AI 生成的初稿

**检测重点**:
- [ ] 高频 AI 词替换（心中一暖、倒吸冷气等）
- [ ] 句式多样化
- [ ] 对话标签丰富化
- [ ] 总结性语句改为展示

**AI 味词表**:
| 词汇 | 建议替换 |
|------|---------|
| 心中一暖 | 心头一热/眼眶微热 |
| 倒吸冷气 | 瞳孔微缩/呼吸一滞 |
| 嘴角勾起 | 唇角微扬/勾起一抹弧度 |
| 眼神复杂 | 目光闪烁/神色莫名 |
| 深吸一口气 | 定了定神/平复心绪 |
| 缓缓 | 慢慢/徐徐 |
| 仿佛 | 好像/如同 |
| 似乎 | 好像/仿佛 |

---

### 模式 4: 扩写模式（大纲→正文）

**适用**: 只有场景大纲时

**输入**:
```
场景 1: 主角醒来，发现穿越
场景 2: 系统觉醒
场景 3: 第一个任务
```

**输出**: 3000-5000 字正文

**扩写技巧**:
- 增加环境描写
- 增加心理活动
- 增加对话互动
- 增加动作细节
- 增加五感体验

---

### 模式 5: 缩写模式（字数超标）

**适用**: 初稿超过目标字数 20%+

**删减优先级**:
1. 冗余环境描写（>200 字）
2. 无意义对话
3. 重复心理描写
4. 无关支线

**保留**:
- 核心情节
- 关键对话
- 伏笔铺垫
- 爽点高潮

---

### 模式 6: 敏感词检测（发布前）

**检测类型**:
- [ ] 政治敏感
- [ ] 色情低俗
- [ ] 暴力血腥
- [ ] 封建迷信
- [ ] 侵权内容

**替换表**:
| 原词 | 替换 |
|------|------|
| 杀人 | 击杀/斩杀 |
| 强奸 | 玷污/侵犯 |
| 自杀 | 自尽/了断 |
| 政府 | 官府/朝廷 |
| 警察 | 捕快/衙役 |

## 润色检查清单

### 文笔检查

- [ ] 没有重复用词（同一段落）
- [ ] 句式有变化（长短句结合）
- [ ] 五感描写充分（至少 3 种）
- [ ] 修辞手法多样（比喻/拟人/排比）
- [ ] 没有 AI 味表达

### 节奏检查

- [ ] 开场有钩子（300 字内）
- [ ] 中段有发展（不注水）
- [ ] 高潮有张力（短句 + 动作）
- [ ] 结尾有悬念（钩子）
- [ ] 段落长短有致

### 对话检查

- [ ] 对话符合人设
- [ ] 对话有信息量
- [ ] 标签多样化（道/问/笑/叹）
- [ ] 有无标签对话
- [ ] 用动作替代标签

### 逻辑检查

- [ ] 时间线连续
- [ ] 地点转换合理
- [ ] 因果关系正确
- [ ] 没有机械降神

## 润色工作流

```
初稿完成
    ↓
选择润色模式
    ↓
自动润色（AI）
    ↓
手动调整（人工）
    ↓
润色后检查
    ↓
进入自检环节（continuity_checklist）
```

## 润色指令模板

### 精修指令

```
请润色以下段落，要求：
1. 增强五感描写（视觉/听觉/嗅觉/触觉）
2. 优化修辞手法（比喻/拟人/排比）
3. 加深情绪渲染
4. 保持原意不变
5. 字数控制在±10%

【待润色段落】
（粘贴内容）
```

### 去 AI 味指令

```
请去除以下段落的 AI 味，要求：
1. 替换高频 AI 词（心中一暖、倒吸冷气等）
2. 句式多样化（避免单调）
3. 对话标签丰富化
4. 总结性语句改为展示
5. 保持原意不变

【待润色段落】
（粘贴内容）
```

### 扩写指令

```
请根据以下场景大纲扩写为正文，要求：
1. 字数：3000-5000 字
2. 对话比例：30-40%
3. 增加环境/心理/动作描写
4. 增加五感体验
5. 保持节奏紧凑

【场景大纲】
（粘贴大纲）
```
