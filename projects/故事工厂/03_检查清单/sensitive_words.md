# 敏感词检测清单

**用途**: 发布前最终检查  
**位置**: 质量门禁之后、导出之前

---

## 检测类型

### 一、政治敏感

| 类型 | 示例 | 处理方式 |
|------|------|---------|
| 国家领导人 | 真实姓名 | 删除/改为虚构 |
| 政治事件 | 真实历史事件 | 改为虚构 |
| 政府机构 | 真实部门 | 改为虚构（官府/朝廷） |
| 地图领土 | 敏感地区 | 改为虚构地名 |

---

### 二、色情低俗

| 类型 | 示例 | 处理方式 |
|------|------|---------|
|  explicit 描写 | 性行为细节 | 删除/改为暗示 |
| 低俗语言 | 露骨对话 | 删除/改写 |
| 性暗示 | 过度暧昧 | 适度保留 |

**替换表**:
| 原词 | 替换 |
|------|------|
| 强奸 | 玷污/侵犯/强行 |
| 做爱 | 同房/欢好/亲密 |
| 裸体 | 赤身/未着寸缕 |
| 乳房 | 胸脯/峰峦 |
| 下体 | 身下/私密处 |

---

### 三、暴力血腥

| 类型 | 示例 | 处理方式 |
|------|------|---------|
| 虐杀描写 | 详细杀戮过程 | 简化/侧面描写 |
| 血腥场面 | 鲜血淋漓 | 改为"鲜血涌出" |
| 酷刑 | 详细刑罚过程 | 删除/改为暗示 |

**替换表**:
| 原词 | 替换 |
|------|------|
| 杀人 | 击杀/斩杀/灭杀 |
| 碎尸 | 身死道消/陨落 |
| 鲜血淋漓 | 鲜血涌出/染红衣襟 |
| 酷刑 | 折磨/刑罚 |

---

### 四、封建迷信

| 类型 | 示例 | 处理方式 |
|------|------|---------|
| 真实宗教 | 真实教派 | 改为虚构 |
| 邪教仪式 | 详细仪式过程 | 删除/简化 |
| 迷信活动 | 算命/占卜 | 改为"推演/占卜"（修仙语境可保留） |

---

### 五、侵权内容

| 类型 | 示例 | 处理方式 |
|------|------|---------|
| 同人作品 | 使用他人 IP | 获得授权/改为原创 |
| 抄袭 | 大段复制 | 删除/重写 |
| 商标品牌 | 真实品牌名 | 改为虚构 |

---

## 检测流程

```
1. 自动检测（工具）
   ↓
2. 人工复核（重点段落）
   ↓
3. 问题记录
   ↓
4. 修改/替换
   ↓
5. 再次检测
   ↓
6. 通过→发布
```

## 常见误判

### 修仙语境可保留

| 词汇 | 说明 |
|------|------|
| 修炼/筑基/金丹 | 修仙体系，可保留 |
| 法术/神通 | 修仙设定，可保留 |
| 宗门/长老 | 修仙组织，可保留 |
| 占卜/推演 | 修仙技能，可保留 |

### 需要修改

| 词汇 | 替换 |
|------|------|
| 真实地名 | 虚构地名 |
| 真实人名 | 虚构人名 |
| 真实品牌 | 虚构品牌 |
