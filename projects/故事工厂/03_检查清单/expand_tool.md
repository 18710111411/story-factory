# 扩写工具（水文助手）

**用途**: 从大纲/场景扩展到正文  
**目标**: 日更万字核心工具

---

## 扩写模式

### 模式 1: 场景扩写（500 字→3000 字）

**输入**:
```
场景 1: 许溪醒来，发现穿越
- 地点：陌生房间
- 情绪：震惊 + 疑惑
- 动作：坐起、环顾四周

场景 2: 系统觉醒
- 触发：许溪疑惑时
- 内容：系统介绍 + 新手任务
- 情绪：惊喜 + 警惕
```

**输出**: 3000 字正文

**扩写技巧**:

#### 1. 环境描写（+300 字）
```
清晨的阳光透过雕花窗棂，洒在许溪脸上。
他睁开眼，映入眼帘的是古色古香的房间...
```

#### 2. 心理活动（+500 字）
```
许溪心中掀起惊涛骇浪。
"这不是我的房间...我这是在哪？"
记忆如潮水般涌入...
```

#### 3. 动作细节（+400 字）
```
他猛地坐起身，却因动作太急，一阵头晕目眩。
伸手扶住额头，指尖触碰到冰凉的皮肤...
```

#### 4. 对话互动（+800 字）
```
"许师弟，你醒了？"
一个慵懒的女声从旁边传来...
```

#### 5. 五感体验（+500 字）
```
空气中弥漫着一股甜腻的香气。
耳边传来远处鸟鸣...
```

### 模式 2: 对话扩写（200 字→1000 字）

**输入**:
```
许溪："这是哪？"
叶飞雨："合欢宗。"
许溪："我怎么会在这里？"
叶飞雨："你喝醉了。"
```

**输出**: 1000 字带互动的对话场景

**扩写技巧**:

#### 1. 添加动作标签
```
许溪猛地转头，看见一个红衣女子斜倚在床头。
"这是哪？"他声音干涩。
```

#### 2. 添加心理活动
```
许溪心中警铃大作。合欢宗？那个传说中的宗门？
```

#### 3. 添加表情描写
```
叶飞雨轻笑一声，眉眼如画："合欢宗。"
```

#### 4. 添加环境穿插
```
窗外传来几声鸟鸣，屋内一片寂静。
```

#### 5. 添加潜台词
```
"你喝醉了。"叶飞雨意味深长地看着他，
言下之意：你最好记住这个说法。
```

### 模式 3: 打斗扩写（300 字→2000 字）

**输入**:
```
许溪与王猛对战，轻松获胜。
```

**输出**: 2000 字精彩打斗

**扩写技巧**:

#### 1. 分镜头描写
```
【开场】
两人对峙，气势碰撞，周围空气仿佛凝固。

【第一回合】
王猛率先出手，长剑化作一道白光...
许溪侧身躲过，反手一掌...

【第二回合】
王猛变招，剑势陡然加快...
许溪不退反进，迎了上去...

【高潮】
两人灵力碰撞，发出轰鸣声...
许溪找到破绽，一击制胜！

【收尾】
王猛倒地，许溪负手而立。
```

#### 2. 添加观众反应
```
台下观众纷纷屏住呼吸。
"好快！"
"这一剑王猛用了八成力吧？"
```

#### 3. 添加心理博弈
```
许溪心中冷笑：这种程度的攻击，也想伤我？
但他表面不动声色，等待最佳时机。
```

#### 4. 添加环境互动
```
剑气划过，地面留下深深的痕迹。
周围的树木被余波震得沙沙作响。
```

### 模式 4: 感情戏扩写（200 字→1500 字）

**输入**:
```
许溪向苏清雪表白，苏清雪脸红答应。
```

**输出**: 1500 字细腻感情戏

**扩写技巧**:

#### 1. 环境烘托
```
夕阳西下，两人站在桃花树下。
花瓣随风飘落，落在苏清雪的发间。
```

#### 2. 微表情描写
```
许溪深吸一口气，声音有些颤抖：
"清雪，我...我喜欢你。"
苏清雪的脸颊以肉眼可见的速度红了起来。
```

#### 3. 肢体语言
```
苏清雪低头绞着衣角，脚尖轻轻踢着地面。
"我...我也喜欢许师兄。"声音细若蚊呐。
```

#### 4. 内心独白
```
许溪心中狂喜，却努力保持平静。
这一世，她终于回应了我的感情。
```

#### 5. 回忆穿插
```
想起前世，她到死都不知道自己的心意。
这一世，他不会再错过。
```

## 扩写指令模板

### 通用扩写指令

```
请根据以下大纲扩写为正文，要求：

【基本信息】
- 目标字数：3000-5000 字
- 对话比例：30-40%
- 文风：轻松/严肃/热血

【扩写要求】
1. 增加环境描写（五感体验）
2. 增加心理活动（内心独白）
3. 增加动作细节（连贯流畅）
4. 增加对话互动（符合人设）
5. 保持节奏紧凑（不注水）

【场景大纲】
（粘贴大纲）

【人物设定】
（粘贴人设）

【前情提要】
（粘贴上章摘要）
```

## 扩写质量检查

| 指标 | 标准 | 检查方法 |
|------|------|---------|
| 字数 | 达到目标±10% | 统计 |
| 对话比例 | 30-40% | 统计 |
| 五感描写 | 至少 3 种 | 检查 |
| 心理活动 | 每场景至少 1 次 | 检查 |
| 动作连贯 | 无跳跃 | 通读 |
| 节奏控制 | 不拖沓 | 通读 |

## 扩写注意事项

### 避免注水

- [ ] 不要重复描写同一事物
- [ ] 不要无意义环境描写（>200 字）
- [ ] 不要废话对话
- [ ] 不要过度心理独白

### 保持节奏

- [ ] 每 500 字有小进展
- [ ] 每 1000 字有小高潮
- [ ] 章末必须有钩子

### 保持一致

- [ ] 人设不 OOC
- [ ] 文风统一
- [ ] 时间线连续
