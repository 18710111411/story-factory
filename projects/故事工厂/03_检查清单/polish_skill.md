# 文章润色技能（全面版）

**用途**: 全面提升文笔质量，不只是去 AI 味  
**使用时机**: 初稿完成后、自检之前

---

## 🎯 润色维度

### 1. 去 AI 味（基础）

**检查项**:
- [ ] 替换高频 AI 词（心中一暖、倒吸冷气、嘴角勾起等）
- [ ] 句式多样化（避免连续使用相同句式）
- [ ] 对话标签丰富化（不要全是"XX 道"）
- [ ] 总结性语句改为展示（Tell→Show）

**AI 味词表**:
| AI 味词汇 | 建议替换 |
|---------|---------|
| 心中一暖 | 心头一热/眼眶微热/心底泛起涟漪 |
| 倒吸冷气 | 瞳孔微缩/呼吸一滞/心头一紧 |
| 嘴角勾起 | 唇角微扬/勾起一抹弧度/唇角轻扬 |
| 眼神复杂 | 目光闪烁/神色莫名/眼底情绪翻涌 |
| 深吸一口气 | 定了定神/平复心绪/调整呼吸 |
| 缓缓 | 慢慢/徐徐/渐渐 |
| 仿佛 | 好像/如同/好似 |
| 似乎 | 好像/仿佛/好似 |

---

### 2. 文笔提升（进阶）

**检查项**:
- [ ] 修辞手法多样（比喻/拟人/排比/对比）
- [ ] 词汇丰富度（避免同一段落重复用词）
- [ ] 节奏控制（长短句结合，张弛有度）
- [ ] 语言风格统一（符合题材和人物设定）

**修辞示例**:

**比喻**:
- 原句：他很生气。
- 润色：他眼中燃起怒火，仿佛要将眼前的一切焚烧殆尽。

**拟人**:
- 原句：风吹过树林。
- 润色：风穿过树林，树叶沙沙作响，像是在低声诉说着什么。

**排比**:
- 原句：他很强，很快，很准。
- 润色：他的剑很快，快如闪电；他的剑很准，准如鹰眼；他的剑很狠，狠如毒蛇。

---

### 3. 可读性优化（结构）

**检查项**:
- [ ] 段落长短有致（避免超长段落）
- [ ] 对话分段清晰（一人一段）
- [ ] 场景转换自然（有时间/地点提示）
- [ ] 信息量适中（不过载、不空洞）

**段落控制**:
- 单句段落：用于强调（每章 3-5 处）
- 短段落（2-3 句）：用于对话和快节奏场景
- 中段落（4-6 句）：用于一般描写
- 长段落（>7 句）：避免，除非必要

---

### 4. 心理描写（深度）

**检查项**:
- [ ] 内心独白自然（不矫情、不做作）
- [ ] 情绪变化有层次（不是突然转变）
- [ ] 心理活动与行为一致（不 OOC）
- [ ] 避免过度心理描写（每章 2-3 处深度心理描写即可）

**心理描写技巧**:

**直接描写**:
> 许溪心中冷笑。陈长生啊陈长生，前世我视你为生死兄弟，今生你却这样对我。

**间接描写（通过行为）**:
> 许溪接过酒杯，一饮而尽。酒入喉，辛辣中带着一丝甜味，像极了他们曾经的兄弟情。

**间接描写（通过生理反应）**:
> 许溪的手指微微颤抖。他握紧拳头，指甲嵌入掌心，渗出丝丝血迹。

---

### 5. 景色描写（氛围）

**检查项**:
- [ ] 五感描写充分（视觉/听觉/嗅觉/触觉/味觉）
- [ ] 景色服务于情绪（不是为写景而写景）
- [ ] 避免过度描写（每处景色 100-200 字即可）
- [ ] 景色与情节呼应（开场/高潮/结尾的景色不同）

**五感描写示例**:

**视觉**:
> 夜空如墨，繁星点点。青云宗的建筑在月光下显得格外壮观。

**听觉**:
> 远处传来几声鸟鸣，更显幽静。风吹过树林，树叶沙沙作响。

**嗅觉**:
> 空气中弥漫着一股甜腻的香气，像是某种花香，又像是……女人的体香。

**触觉**:
> 许溪扶着额头，指尖触碰到冰凉的皮肤，让他清醒了几分。

**味觉**:
> 酒入喉，辛辣中带着一丝甜味，像极了他们曾经的兄弟情。

---

### 6. 外貌描写（人物）

**检查项**:
- [ ] 外貌符合人设（性格/身份/年龄）
- [ ] 避免过度描写（首次登场详细，后续简略）
- [ ] 外貌变化有原因（受伤/突破/情绪变化）
- [ ] 避免脸谱化（不是所有美女都"眉眼如画"）

**外貌描写技巧**:

**首次登场（详细）**:
> 叶飞雨斜倚在床头，衣衫半敞，露出大片雪白的肌肤。她眉眼如画，唇角含笑，眼神里却带着几分玩味。一袭红衣似火，衬得她肌肤如雪，整个人如同盛开的彼岸花，美艳中带着危险。

**后续登场（简略）**:
> 叶飞雨还是那身红衣，还是那么美，还是那么神秘。

**情绪变化时的外貌**:
> 叶飞雨脸色苍白，原本红润的嘴唇失去了血色。她的眼神黯淡无光，仿佛一夜之间老了十岁。

## 🛠️ 润色指令模板

### 完整润色指令

```
请润色以下小说段落，要求：

【润色维度】
1. 去 AI 味：替换高频 AI 词，句式多样化
2. 文笔提升：增加修辞手法，丰富词汇
3. 可读性：优化段落结构，对话分段清晰
4. 心理描写：增加内心独白和情绪层次
5. 景色描写：增加五感体验，营造氛围
6. 外貌描写：符合人设，避免脸谱化

【原文】
（粘贴需要润色的段落）

【题材类型】
玄幻/都市/言情/仙侠

【人物设定】
（简要说明主要人物的性格和身份）

【本章情绪基调】
轻松/紧张/悲伤/热血
```

## 📋 润色检查清单

### 润色前检查

- [ ] 初稿已完成
- [ ] 字数符合要求
- [ ] 情节逻辑通顺
- [ ] 无明显错别字

### 润色后检查

- [ ] AI 味词汇已替换
- [ ] 修辞手法多样
- [ ] 段落长短有致
- [ ] 心理描写有深度
- [ ] 景色描写有氛围
- [ ] 外貌描写符合人设
- [ ] 可读性良好

---

## 🎯 润色质量评估

| 维度 | 评分标准 | 及格线 |
|------|---------|--------|
| 去 AI 味 | AI 词<3 个/章 | 5/5 |
| 文笔提升 | 修辞手法≥3 种/章 | 4/5 |
| 可读性 | 段落长短有致 | 5/5 |
| 心理描写 | 深度描写≥2 处/章 | 4/5 |
| 景色描写 | 五感描写≥3 种/章 | 4/5 |
| 外貌描写 | 符合人设，不脸谱化 | 5/5 |

**总分**: 27/30 为优秀，24/30 为良好，<24/30 需重新润色

## 📝 润色流程

```
1. 写完初稿
   ↓
2. 通读一遍，标记需要润色的段落
   ↓
3. 使用润色指令（AI 辅助）
   ↓
4. 检查润色结果，手动调整
   ↓
5. 再次通读，确认流畅
   ↓
6. 进入自检环节（continuity_checklist）
```
