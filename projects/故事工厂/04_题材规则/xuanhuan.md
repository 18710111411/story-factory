# 玄幻题材规则

**适用**: 东方玄幻、高武世界

---

## 核心要素

- [ ] 修炼体系明确（境界名称、差距）
- [ ] 金手指/系统合理
- [ ] 爽点密集（打脸/收获/成长）
- [ ] 世界观宏大（多地图、多势力）

---

## 修炼体系

### 常见境界
```
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫
```

### 境界差距
- 小境界：1-2 倍差距
- 大境界：10 倍 + 差距
- 越级挑战：需要特殊原因（功法/法宝/燃烧精血）

---

## 爽点要求

| 周期 | 爽点类型 | 数量 |
|------|---------|------|
| 每章 | 小爽点 | 1-2 个 |
| 每 3 章 | 中爽点 | 1 个 |
| 每 10 章 | 大爽点 | 1 个 |
| 每卷 | 超大爽点 | 1 个 |

### 爽点类型
- 打脸型：被嘲讽→反击→震惊
- 收获型：获得宝物/功法/奖励
- 成长型：突破/领悟/晋升
- 守护型：保护重要的人
- 装逼型：低调展示实力

---

## 节奏要求

- **每章字数**: 3000-5000 字
- **对话比例**: 20-40%
- **开场**: 300 字内必须有钩子
- **章末**: 必须有悬念钩子

---

## 禁忌

- [ ] 不要过度虐主
- [ ] 不要圣母心泛滥
- [ ] 不要战力崩坏
- [ ] 不要后宫无脑收
- [ ] 不要重复套路

---

## 审计维度

| 维度 | 检查点 |
|------|--------|
| 修炼体系 | 境界名称、差距合理 |
| 金手指 | 有限制、有代价 |
| 爽点密度 | 达标 |
| 战力系统 | 不崩坏 |
| 世界观 | 自洽 |
| 人物动机 | 合理 |
