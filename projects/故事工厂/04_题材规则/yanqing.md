# 言情题材规则

**适用**: 古代言情、现代言情

---

## 核心要素

- [ ] 感情发展有铺垫
- [ ] 主角人设有魅力
- [ ] 冲突合理不降智
- [ ] 情感细腻真实

---

## 感情线设计

### 发展阶段
```
初遇 → 相识 → 暧昧 → 误会 → 和解 → 表白 → 在一起
```

### 情感拉扯
- 推拉有度
- 误会合理（不强行误会）
- 和解有铺垫

### 甜虐比例
- 甜宠文：80% 甜 +20% 虐
- 虐恋文：40% 甜 +60% 虐
- 爽文：90% 甜 +10% 虐

---

## 人设要求

### 主角
- 有魅力（外貌/性格/能力）
- 有成长
- 有缺点但不致命

### 配角
- 不降智
- 有独立人格
- 不工具人

### 反派
- 动机合理
- 不脸谱化
- 有苦衷（可选）

---

## 节奏要求

- **每章字数**: 2500-4000 字
- **对话比例**: 40-60%
- **感情进展**: 每 3 章有小进展
- **章末**: 情感钩子/悬念

---

## 禁忌

- [ ] 不要工业糖精
- [ ] 不要强行误会
- [ ] 不要配角降智
- [ ] 不要感情突变
- [ ] 不要油腻描写

---

## 审计维度

| 维度 | 检查点 |
|------|--------|
| 感情发展 | 有铺垫 |
| 人设魅力 | 主角讨喜 |
| 配角智商 | 不降智 |
| 情感真实 | 不矫情 |
| 对话质量 | 符合人设 |
| 节奏控制 | 不拖沓 |
