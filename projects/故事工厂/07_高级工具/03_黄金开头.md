# 黄金开头生成器

**用途**: 生成吸睛的开头（前 3 章）  
**目标**: 3 章内抓住读者

---

## 黄金三章原则

### 第 1 章：金手指觉醒

**必须包含**:
- [ ] 主角登场（身份/处境）
- [ ] 核心冲突出现
- [ ] 金手指觉醒
- [ ] 章末钩子

**开场方式**:

| 方式 | 示例 | 适用 |
|------|------|------|
| 对话开场 | "退婚？好，我签！" | 快节奏文 |
| 动作开场 | 许溪猛地睁开眼，从床上坐起 | 穿越/重生 |
| 环境开场 | 清晨的阳光透过雕花窗棂 | 慢热文 |
| 倒叙开场 | 三百年后，许溪站在巅峰 | 回忆式 |
| 悬念开场 | 许溪死了，又活了 | 重生/系统 |

### 第 2 章：展示能力

**必须包含**:
- [ ] 金手指能力展示
- [ ] 第一个小爽点
- [ ] 引入重要配角
- [ ] 章末钩子

### 第 3 章：首次冲突

**必须包含**:
- [ ] 首次打脸/冲突
- [ ] 读者期待感建立
- [ ] 引出后续剧情
- [ ] 章末钩子

## 常见开头问题

### 问题 1: 开场太慢

**表现**: 前 500 字还在环境描写  
**解决**: 第一段就进入冲突

### 问题 2: 金手指太晚

**表现**: 第 1 章结束还没觉醒  
**解决**: 前半章就觉醒

### 问题 3: 没有钩子

**表现**: 章末平淡，不想点下一章  
**解决**: 章末必须留悬念

### 问题 4: 信息过载

**表现**: 第一章塞太多设定  
**解决**: 设定分散到后续章节
