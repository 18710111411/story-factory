# 创作规则 (Book Rules)

## 题材规则
- 题材：修仙/重生/系统
- 核心梗：260 年后的合欢老祖重生，获得社死系统——尴尬=变强
- 对标作品：《大奉打更人》《重生之都市修仙》《我师兄实在太稳健了》

## 主角设定
- 姓名：许溪（前世林默）
- 性格：老谋深算 + 谨慎多疑 + 乐子人 + 护短
- 金手指：社死系统（社死→积分→奖励）
- 核心动机：前世复仇 + 今生守护 + 超脱轮回

## 数值上限
- 修炼境界上限：超脱（300 章）
- 年龄跨度：前世 260 岁 + 今生 16 岁
- 战力上限：不越级挑战（除非燃烧精血/底牌尽出）

## 自定义禁令
- 禁止出现：真实地名/人名/品牌
- 尽量避免：过度虐主、圣母心泛滥
- 谨慎使用：后宫无脑收、重复套路

## 节奏要求
- 每章字数：4000-6000 字
- 爽点密度：每章 1-2 个小爽点
- 对话比例：25-35%
- 章末钩子：必须有（新危机/新发现/反转/悬念）

## 文风要求
- 叙事视角：第三人称限知（主要跟随许溪视角）
- 语言风格：轻松幽默 + 热血战斗
- 禁用词汇：心中一暖→心头一热、倒吸冷气→瞳孔微缩、嘴角勾起→唇角微扬

## 题材专属规则

### 修仙规则
- [x] 修炼体系明确（炼气→筑基→金丹→元婴→化神→超脱）
- [x] 境界差距合理（小境界 1-2 倍，大境界 10 倍+）
- [x] 不越级挑战（除非特殊原因）
- [x] 功法/法宝与设定一致
- [x] 宗门/势力名称虚构
