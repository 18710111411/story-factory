## 二、核心公式提炼

### 核心梗公式

```
核心梗 = 重生老祖 + 社死系统 + 信息优势 + 乐子人

示例：
"260 年后的合欢宗老祖林默，重生回到 300 年前的青云宗许溪身上。
获得'社死系统'——只要让人尴尬就能变强！

前世被正道盟和身边人背叛，今生不再信任何人；
开局被捉奸在床，被迫负责合欢宗师姐；
外门大比跳舞夺冠，震惊整个青云宗；
正魔大战布局天下，揭开天道阴谋……

这一世，他要改变所有遗憾，超脱轮回！
```

---

### 金手指公式

| 项目 | 设定 |
|------|------|
| 类型 | 社死系统 |
| 获取方式 | 重生时意外获得 |
| 核心能力 | 社死→积分→奖励 |
| 限制条件 | 必须被动社死，不能主动作死 |
| 升级方式 | 社死值积累 |

**奖励类型**:
- 灵石/丹药（基础资源）
- 功法/秘籍（修炼辅助）
- 线索/信息（关键剧情）
- 特殊道具（稀有）

---

### 爽点公式

| 周期 | 爽点类型 | 数量 | 说明 |
|------|---------|------|------|
| 每章 | 小爽点 | 1-2 个 | 社死 + 小收获 |
| 每 3 章 | 中爽点 | 1 个 | 小打脸/小高潮 |
| 每 10 章 | 大爽点 | 1 个 | 中型冲突解决 |
| 每卷 | 超大爽点 | 1 个 | 卷终高潮 |

**爽点类型**:
- 社死型：尴尬场面 + 无奈接受
- 打脸型：被嘲讽→反击→震惊
- 收获型：获得宝物/奖励/能力
- 成长型：突破/领悟/晋升
- 守护型：保护重要的人
- 装逼型：低调展示实力

## 三、可借鉴点总结

### 从《大奉打更人》借鉴
1. 穿越者身份反差（老祖 vs 小弟子）
2. 利用信息优势装逼打脸
3. 社死梗增加趣味性
4. 探案/阴谋主线

### 从《重生之都市修仙》借鉴
1. 仙尊重生的绝对实力碾压
2. 家人/宗门被威胁的守护动机
3. 连续打脸节奏快
4. 扮猪吃虎爽点密集

### 从《我师兄实在太稳健了》借鉴
1. 极致谨慎人设（反差萌）

## 四、需避免点

1. **后期战力崩坏** - 提前规划修炼体系上限
2. **感情线过于分散** - 控制女主数量（2-3 个核心）
3. **重复打脸套路** - 设计多样化爽点
4. **节奏过慢** - 前 10 章必须有第一个大高潮
5. **人设崩塌** - 保持主角谨慎 + 乐子人人设

---

## 五、本书差异化设计

### 核心差异化

| 元素 | 传统套路 | 本书设计 |
|------|---------|---------|
| 重生 | 都市/家族 | 修仙界 + 老祖身份 |
| 系统 | 任务/签到 | 社死得奖励（创新） |
| 爽点 | 打脸/收获 | 社死尴尬 + 修为提升 |
| 人设 | 龙傲天/苟道 | 老祖 + 乐子人 + 谨慎 |
| 感情 | 多女主后宫 | 有舍有得（部分退出） |

### 创新点

1. **社死系统** - 尴尬=变强，创新金手指
2. **老祖重生** - 元婴老祖 vs 炼气弟子，反差大
3. **乐子人主角** - 不是苦大仇深，是乐子人
4. **部分女主退出** - 不是所有都陪到最后，更真实
