# 喜磨坊早餐 · 角色档案

---

## 老李（李建国）

| 属性 | 描述 |
|------|------|
| **年龄** | 45 岁 |
| **外貌** | 身体微胖，圆脸，常穿白色围裙，头发有点自然卷 |
| **性格** | 热心肠、说话实在、嘴笨心热、不善表达关心 |
| **背景** | 下岗后开了这家早餐店，把顾客当家人 |
| **口头禅** | "少废话"、"这就够了"、"说啥说" |
| **动作习惯** | 搓手、擦汗、揉面时思考 |
| **情感线** | 从"老板"逐渐成为"大家长" |

**角色弧光**：
- 第 1 集：破例录用小张（展现善良）
- 第 2 集：主动接孩子（行动派关怀）
- 第 3 集：调整排班、提供学习空间（深度关怀）

## 小张（张小梅）

| 属性 | 描述 |
|------|------|
| **年龄** | 38 岁 |
| **外貌** | 朴素干净，马尾辫，眼神坚毅，常穿浅色衣服 |
| **性格** | 勤劳朴实、谨慎、感恩、坚韧 |

**角色弧光**：
- 第 1 集：紧张面试，意外被录用（希望）
- 第 2 集：孩子生病被帮助（感动）
- 第 3 集：被关怀，找到家的感觉（归属）

## 表演指导

### 老李
- **语气**：低沉、实在、不啰嗦
- **表情**：严肃但眼神温和
- **肢体**：动作干脆，不拖泥带水
- **关键点**：关心时不直视对方（害羞）

### 小张
- **语气**：轻柔、谨慎、带点口音
- **表情**：常带感激、眼眶易红
- **肢体**：微微弯腰（谦卑）、手不自觉搓动
- **关键点**：情感爆发时克制（坚强）

### 汪大妈
- **语气**：响亮、直接、带点抱怨
- **表情**：皱眉但眼神关切
- **肢体**：动作幅度大、喜欢拍人
- **关键点**：关心时用"抱怨"包装
