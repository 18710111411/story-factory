## 使用指南

### 即梦平台设置

| 参数 | 建议值 |
|------|--------|
| **时长** | 按各镜头指定时长设置 |
| **比例** | 16:9 横屏 |
| **分辨率** | 1920×1080 或更高 |
| **帧率** | 24fps 或 30fps |
| **生成顺序** | 按镜头 1→18 依次生成 |

### 对话录制建议

1. **后期配音**：生成视频后，找配音演员录制对话
2. **AI 配音**：使用 Azure TTS/ElevenLabs 生成中文配音
3. **语速建议**：
   - 老李：低沉实在，语速中等（180 字/分钟）
   - 小张：轻柔温和，略带口音（170 字/分钟）
   - 汪大妈：响亮直接，语速较快（200 字/分钟）

### 口型同步

- 使用 Wav2Lip 或类似工具同步口型
- 或在即梦平台使用"口型同步"功能（如支持）
- 对话时长已在镜头时长内预留充足时间
