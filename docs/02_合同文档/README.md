## 🔄 合同管理流程

### 合同起草
1. 使用合同模板
2. 填写待填写项
3. 内部审核

### 合同审核
1. 双方法务审核
2. 修改完善
3. 确认最终版本

### 合同签署
1. 打印两份
2. 双方签字盖章
3. 各执一份

### 合同归档
1. 扫描电子版
2. 存入 `已签署合同/` 目录
3. 更新本文档

### 合同执行
1. 按合同执行项目
2. 记录执行进度
3. 定期回顾
