# HEARTBEAT.md - 定期检查任务

# 每日检查 (建议每天运行 2-4 次)
- [ ] 检查系统健康状态 (healthcheck)
- [ ] 查看 GitHub 通知 (PR/Issue)
- [ ] 检查未读邮件摘要
- [ ] 查看日历事件 (未来 24 小时)
- [ ] 检查节点连接状态

# 每周检查 (每周一运行)
- [ ] 更新所有技能 (clawhub update --all)
- [ ] 检查 OpenClaw 版本更新
- [ ] 审查 MEMORY.md 内容

# 使用说明
# - 取消注释要启用的任务
# - 心跳检查频率由 gateway 配置决定
# - 使用 `clawhub list` 查看已安装技能
