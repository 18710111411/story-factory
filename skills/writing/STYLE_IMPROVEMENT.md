# 文笔提升指南 - 避免流水账与提升文学性

> **目的**：解决 AI 写作中常见的"流水账"问题，提升文笔质量和文学感染力
> **适用对象**：AI 写作助手、人类作者、人机协作场景
> **核心理念**：好故事不是记录发生了什么，而是让读者感受到发生了什么

---

## A. 流水账问题诊断

### A1. 什么是流水账？

**定义**：机械地按时间顺序记录事件，缺乏重点、情感和文学性的叙述方式

### 流水账的典型特征

```markdown
## 流水账识别清单 □

### 结构特征
- [ ] 按时间顺序平铺直叙（"然后...然后...然后..."）
- [ ] 每件事篇幅相近，无详略区分
- [ ] 缺乏场景聚焦，频繁跳跃
- [ ] 开头结尾模式化（"早上起床"、"一天结束了"）

### 语言特征
- [ ] 大量使用"我做了 X"、"他去了 Y"句式
- [ ] 动词单一（走、说、看、想重复出现）
- [ ] 缺乏细节描写，只有事件骨架
- [ ] 形容词/副词贫乏或滥用

### 情感特征
- [ ] 读者无法感受到人物情绪
- [ ] 重要时刻缺乏情感渲染
- [ ] 人物反应模式化（"很高兴"、"很伤心"）
- [ ] 缺乏内心独白和心理描写

### 节奏特征
- [ ] 所有事件叙述速度相同
- [ ] 紧张时刻不紧张，舒缓时刻不舒缓
- [ ] 缺乏停顿和留白
- [ ] 高潮部分不够突出

### 示例对比

**流水账版本**：
> 早上我起床了。然后我刷牙洗脸。然后我吃早餐。然后我去学校。
> 在学校我见到了小明。我和小明说话。然后上课了。老师讲课。
> 然后放学了。我回家。然后我写作业。然后我睡觉了。

**优化版本**：
> 闹钟响第三遍时，我终于把自己从床上拽起来。卫生间镜子里
> 的脸浮肿得陌生，牙刷在嘴里机械地来回，薄荷味刺得眼睛发酸。
> 校门口，小明老远就挥手，那副傻笑和三年前一模一样。"你迟到了
> 啊。"他递来热腾腾的包子，白气在冬日的空气里散开。上课铃响时，
> 我突然想起昨晚的梦——梦里也是这样的冬天，也是这样的包子，
> 只是递包子的人，已经不在了。
```

### A2. 为什么 AI 容易写流水账？

```markdown
## AI 流水账成因分析

### 技术原因
1. **概率生成特性**
   - AI 倾向于选择最安全、最常见的表达
   - 避免冒险使用复杂修辞
   - 优先保证逻辑正确而非文学性

2. **上下文窗口限制**
   - 难以维持长距离的伏笔和呼应
   - 倾向于局部连贯而非整体节奏
   - 每次生成只关注当前段落

3. **训练数据偏差**
   - 大量网络小说本身就是流水账
   - 优秀文学作品的比例较低
   - 模式化表达更容易被学习

### 使用原因
1. **提示词不当**
   - 只告诉 AI"写什么"，没说"怎么写"
   - 缺乏风格指引和情感要求
   - 没有设定文学性标准

2. **迭代不足**
   - 接受第一次输出，不做修改
   - 缺乏人工润色环节
   - 没有针对性重写指令

3. **期望错位**
   - 期望 AI 一次性产出高质量文本
   - 忽视 AI 作为辅助工具的定位
   - 不理解 AI 需要引导和约束
```

### A3. 流水账 vs 好故事

```markdown
## 流水账与好故事的本质区别

| 维度 | 流水账 | 好故事 |
|------|--------|--------|
| **焦点** | 发生了什么 | 人物如何感受 |
| **结构** | 时间顺序 | 情感/冲突驱动 |
| **细节** | 均匀分布 | 重点突出 |
| **节奏** | 一成不变 | 快慢有致 |
| **语言** | 陈述事实 | 营造体验 |
| **读者感受** | "知道了" | "感受到了" |
| **记忆点** | 几乎没有 | 场景/台词/情感 |
| **重读价值** | 低 | 高 |

### 核心差异图解

```
流水账：事件 A → 事件 B → 事件 C → 事件 D
         （平等对待，无重点）

好故事：铺垫 A → 发展 B → 高潮 C → 余韵 D
         （重点突出，情感曲线）
         ╰──── 情感强度 ────╯
```

### 判断标准

**问自己这三个问题**：
1. 读者读完这段会记住什么？
2. 这段文字能让读者感受到什么情绪？
3. 如果删掉这段，故事会受影响吗？

如果答案都是"没什么"，那就是流水账。

---
```

## B. 避免流水账的方法

### B1. 场景选择原则

```markdown
## 什么值得写，什么可以跳过

### 值得详写的场景（占篇幅 70-80%）

**高情感强度**：
- [ ] 人物面临重大选择
- [ ] 关系发生转变（相遇/决裂/和解）
- [ ] 秘密被揭示
- [ ] 理想与现实冲突
- [ ] 失去或获得重要事物

**高冲突强度**：
- [ ] 正面对抗（言语/肢体）
- [ ] 内心挣扎
- [ ] 时间压力下的决策
- [ ] 多方利益博弈
- [ ] 道德困境

**高信息密度**：
- [ ] 关键伏笔埋设/回收
- [ ] 世界观重要揭示
- [ ] 人物背景曝光
- [ ] 情节重大转折
- [ ] 主题集中表达

**高象征意义**：
- [ ] 重复出现的意象
- [ ] 具有隐喻性的场景
- [ ] 人物成长的标志事件
- [ ] 命运转折点

### 可以略写的场景（占篇幅 20-30%）

**过渡性场景**：
- [ ] 人物移动（用一句话带过）
- [ ] 日常例行活动
- [ ] 信息传递过程
- [ ] 时间流逝

**低价值场景**：
- [ ] 重复性事件（第二次类似经历）
- [ ] 无冲突的日常对话
- [ ] 纯功能性场景（只为交代信息）
- [ ] 读者可自行脑补的内容

### 场景取舍决策树

```
这个场景是否...
    │
    ├─→ 推动情节？──否──→ [跳过/一句话带过]
    │       │
    │       是
    │       ↓
    ├─→ 揭示人物？──否──→ [考虑删除]
    │       │
    │       是
    │       ↓
    ├─→ 有情感强度？──否──→ [略写]
    │       │
    │       是
    │       ↓
    └─→ 有高潮/转折？──是──→ [详写，占主要篇幅]
```

### 场景篇幅分配建议

```
单章（5000 字）建议分配：
┌─────────────────────────────────────┐
│  核心场景（1 个）    │  2500-3000 字  │  50-60%  │
│  次要场景（1-2 个）  │  1500-2000 字  │  30-40%  │
│  过渡场景（若干）    │  500 字       │  10%    │
└─────────────────────────────────────┘

### B2. 详略得当的技巧

```markdown
## 详写与略写的具体技巧

### 详写技巧（慢镜头）

**1. 时间拉伸法**
将几秒钟的事件写成几百字：
```
原文（略写）：他推开门，走进房间。

详写版本：
他的手握住门把，金属的凉意顺着掌心蔓延。
停顿了一秒——也许更久——然后缓缓转动。
"咔哒"一声，锁舌弹开。门轴发出轻微的呻吟，
像是抗议这久违的打扰。他深吸一口气，
踏进门槛，灰尘在斜射的阳光里翻滚，
像是被惊醒的往事。
```

**2. 多感官展开法**
```
事件：她喝咖啡

展开：
视觉：深褐色的液体在杯子里旋转，表面浮着细密的泡沫
嗅觉：焦香混着奶甜味钻进鼻腔
味觉：第一口烫得舌尖发麻，随后是醇厚的苦
触觉：瓷杯的温暖透过掌心，驱散指尖的寒意
听觉：吞咽时轻微的声响，勺子碰杯壁的清脆
```

**3. 内心活动插入法**
```
外部动作：他签字

插入内心：
笔尖悬在纸上，墨水在纸纤维上晕开一个小点。
这一笔下去，三年的婚姻就完了。他想起婚礼那天，
她手抖得戒指都戴不进去，现在轮到他的手抖了。
"签吧。"律师说。他落下名字，比想象中轻，
像一片羽毛落在墓碑上。
```

### 略写技巧（快进）

**1. 概括叙述法**
```
详写：每天早上他六点半起床，刷牙三分钟，
洗脸两分钟，换衣服五分钟，然后下楼买早餐...

略写：日子像复印机里出来的纸，一张张相同。
六点半起床，洗漱，出门，买早餐，上班。
重复了七百三十天，直到那天早上...
```

**2. 时间跳跃法**
```
详写：第一天...第二天...第三天...

略写：一周过去了。/ 转眼到了秋天。/ 那之后，
他们再也没见过面。
```

**3. 留白法**
```
详写：他说了很多安慰的话，她也哭了很久...

略写：他说了什么，她没听清。只记得离开时，
天已经黑了。/ 后面的事，不必细说。
```

### 详略转换信号词

**详写信号**：
- 突然、就在这时、那一刻、刹那间
- 他注意到、她感觉到、他意识到
- 时间仿佛静止、空气凝固

**略写信号**：
- 日子一天天过去、转眼、很快
- 总之、长话短说、简而言之
- 之后的事不重要、不必细说

---
```

### B3. 节奏控制方法

```markdown
## 节奏快慢结合的技巧

### 节奏类型识别

| 节奏类型 | 适用场景 | 字数/事件比 | 句式特征 |
|----------|----------|-------------|----------|
| **极慢** | 高潮瞬间、情感爆发 | 1000 字/1 分钟 | 长句、多修饰、多停顿 |
| **慢** | 重要对话、心理描写 | 500 字/5 分钟 | 中长句、细节丰富 |
| **中** | 一般叙述、场景转换 | 300 字/30 分钟 | 句式多变 |
| **快** | 动作场景、紧张时刻 | 200 字/1 小时 | 短句、少修饰 |
| **极快** | 时间跳跃、过渡 | 50 字/数天 | 概括句、总结句 |

### 节奏控制技巧

**1. 句子长度控制**
```
快节奏（紧张场景）：
他跑。楼梯在脚下呻吟。脚步声逼近。
不能停。不能回头。转角。撞上人。
道歉。继续跑。肺要炸了。

慢节奏（情感场景）：
她坐在那里，看着窗外的雨一点点打湿
玻璃，然后汇聚成细流，蜿蜒而下，
像是天空也在替她流泪。她想起很多年前
的那个下午，也是这样的雨，也是这样的
等待，只是那个人，再也不会回来了。
```

**2. 段落长度控制**
```
快节奏：多分段，甚至一句一段
他听见了脚步声。
停住。
呼吸。
有人在附近。

慢节奏：长段落，连贯叙述
她花了一下午整理那些旧照片，一张一张地看，
一张一张地回忆，时光在指尖流淌，从青春年少
到中年白发，那些笑过的人，爱过的人，伤害过
的人，都定格在小小的相框里，成了可以触摸的
历史。
```

**3. 信息密度控制**
```
快节奏：信息稀疏，只保留动作
他冲进去。开枪。倒下。离开。

慢节奏：信息密集，加入细节
他踹开门，木屑飞溅。枪口在昏暗的光线里
闪着冷光，食指扣下扳机的瞬间，时间仿佛
被拉长，他能看见子弹旋转着飞出枪膛，
看见目标脸上的表情从惊愕到痛苦，看见
鲜血在空中画出弧线，看见那人缓缓倒下，
像一棵被砍断的树。
```

### 节奏曲线设计

```
标准章节节奏曲线：

情感
强度
  │     ╭───╮         ╭────╮
  │    ╱     ╲       ╱      ╲
  │   ╱       ╲     ╱        ╲      ╭── 结尾
  │  ╱         ╲   ╱          ╲    ╱   （悬念）
  │ ╱           ╲ ╱            ╲  ╱
  │╱             ╰              ╰
  └──────────────────────────────────→
   开场  发展  缓冲  上升  高潮  回落
  （慢）（中）（慢）（快）（极快）（慢）
```

### 节奏检查清单

- [ ] 开场是否吸引注意力（中等节奏）
- [ ] 发展部分是否有起伏（快慢交替）
- [ ] 高潮部分是否足够紧张（快节奏）
- [ ] 高潮后是否有缓冲（慢节奏）
- [ ] 结尾是否留有余韵（慢节奏）
- [ ] 整体是否有节奏变化（避免单调）

```

### B4. 冲突与张力设计

```markdown
## 让场景"活起来"的冲突设计

### 冲突层次

**表层冲突**（可见的）：
- 人物间的争吵/对抗
- 人物与环境的斗争
- 任务/目标的阻碍

**深层冲突**（隐藏的）：
- 人物内心矛盾（想要 vs 应该）
- 价值观冲突（对 vs 对）
- 身份认同危机（我是谁）

### 张力构建技巧

**1. 时间压力法**
```
无张力：他需要找到钥匙开门。

有张力：
距离炸弹爆炸还有三分钟。钥匙就在某个
抽屉里，但他不知道是哪个。十个抽屉，
三分钟，平均每个十八秒。第一个抽屉——
没有。二十秒过去了。第二个——还是没有。
手开始出汗。四十五秒。
```

**2. 信息差法**
```
无张力：他不知道她在说谎。

有张力：
她说"我从来没去过那里"，眼神清澈得像
初雪。他知道她在说谎——因为他口袋里
就装着那张她在场的照片。但他什么也没说，
只是笑了笑，继续听她编下去。有时候，
真相不如谎言有趣。
```

**3. 两难选择法**
```
无张力：他决定救她。

有张力：
左边是女儿，右边是妻子。洪水正在上涨，
他只能救一个。二十年的婚姻，五年的父女。
任何选择都是罪。时间一秒一秒流逝，水
漫过膝盖。他闭上眼睛，听见自己心碎的
声音。
```

**4. 悬念设置法**
```
无张力：他打开门，看见一个人。

有张力：
门把转动的时候，他后悔了。不该打开这扇门，
不该回应那个声音，不该回到这个地方。但
已经晚了。门开了，黑暗里站着一个影子。
影子说话了，声音熟悉得让他浑身冰冷。
"你终于回来了。"影子说，"我等你很久了。"
```

### 张力检测清单

每写完一个场景，问：
- [ ] 读者会担心结果吗？
- [ ] 人物有失去的风险吗？
- [ ] 有未解决的问题吗？
- [ ] 有期待的答案吗？
- [ ] 有延迟的满足吗？

如果都是"是"，张力合格。

### 冲突升级模式

```
小冲突 → 中冲突 → 大冲突 → 最终冲突
  │        │        │         │
[试探]  [升级]  [对抗]   [决战]
  │        │        │         │
[章 1-3] [章 4-6] [章 7-9]  [章 10]

每章内部也应有小冲突曲线

## C. 文笔提升技巧

### C1. 展示而非讲述（Show, Don't Tell）

```markdown
## "展示而非讲述"实操指南

### 核心原则

**讲述（Tell）**：直接告诉读者结论
**展示（Show）**：让读者通过细节自己得出结论

### 常见转换示例

| 讲述（避免） | 展示（推荐） |
|--------------|--------------|
| 他很生气 | 他攥紧拳头，指节发白，牙关咬得腮帮鼓起 |
| 她很伤心 | 她盯着手机屏幕，眼泪滴在上面，模糊了那条"我们分手吧" |
| 天气很热 | 柏油路面蒸腾着热气，知了叫得撕心裂肺，他的衬衫后背湿了一片 |
| 他很紧张 | 他不停看表，手心在裤子上擦了又擦，喉咙发干 |
| 房间很乱 | 衣服堆在椅子上，外卖盒在桌上排成一排，垃圾桶满得溢出来 |
| 她很美 | 她走进来时，整个包间安静了三秒 |
| 他很老 | 他脸上的皱纹深得能藏住一生的故事 |
| 她很害怕 | 她缩在角落，膝盖抵着胸口，呼吸浅得像怕惊动什么 |

### 转换公式

```
讲述：[情绪/状态形容词]
  ↓
展示：[身体反应] + [行为细节] + [环境反应] + [他人反应]
```

### 实操练习

**练习 1：情绪展示**
```
题目：展示"愤怒"，但不出现"愤怒"、"生气"、"怒"等字

示例：
他把杯子摔在地上，玻璃渣溅了一地。"出去。"
他说，声音低得几乎听不见，但每个字都像
冰锥。她站着没动，看着他脖子上的青筋
一跳一跳，像要爆裂的血管。
```

**练习 2：关系展示**
```
题目：展示"他们相爱"，但不直接说

示例：
他自然地接过她手里的包，左手牵住她的右手，
十指相扣。过马路时，他往车流方向站了一步。
她说话时，他看着她，眼神专注得像在听世界上
最重要的事。红灯还有三十秒，他问："冷吗？"
她说："不冷。"他说："手这么凉。"然后把她
的手揣进自己口袋。
```

**练习 3：氛围展示**
```
题目：展示"压抑"，但不出现"压抑"

示例：
会议室里没人说话，只有空调的嗡嗡声。老板
坐在长桌尽头，手指一下一下敲着桌面。每个人
都盯着面前的笔记本，假装在看，其实屏幕上是
空的。有人清了清嗓子，声音在寂静里炸开，
吓得他自己都抖了一下。
```

### 展示过度警示

⚠️ **注意**：不是所有地方都需要展示

**适合讲述的情况**：
- 过渡性信息
- 读者已知内容
- 不重要的细节
- 需要快速推进时

**平衡原则**：
```
关键场景：80% 展示 + 20% 讲述
一般场景：50% 展示 + 50% 讲述
过渡场景：20% 展示 + 80% 讲述

### C2. 五感描写训练

```markdown
## 五感描写全方位训练

### 视觉描写

**层次递进**：
1. 整体印象（颜色、形状、大小）
2. 细节特征（纹理、光泽、状态）
3. 动态变化（移动、变形、交互）
4. 主观感受（联想、情感投射）

**示例**：
```
层次 1：那是一辆旧自行车。
层次 2：那是一辆生锈的永久牌自行车，车漆
        剥落得像皮肤病。
层次 3：链条每转一圈都发出痛苦的呻吟，
        车把上的黑胶布裂开，露出里面的铁。
层次 4：这车比他还老，载过他父亲，载过他，
        现在载着他的儿子。三代人的重量压在
        两根细辐条上，它们居然还没断。
```

### 听觉描写

**声音分类**：
- 人声（说话、呼吸、心跳）
- 自然声（风雨、鸟虫、水流）
- 机械声（车辆、电器、工具）
- 物体声（碰撞、摩擦、破裂）

**描写维度**：
- 音量（震耳欲聋/细若游丝）
- 音调（尖锐/低沉）
- 节奏（急促/缓慢）
- 质感（清脆/沉闷）
- 距离（近在耳边/遥远模糊）

**示例**：
```
劣质版：外面很吵。

优质版：
楼下传来收废品的喇叭声，嘶哑的女声喊着
"旧冰箱旧洗衣机"，尾音拖得老长。隔壁
装修的电钻像是要钻穿他的脑浆，一下一下。
远处有汽车鸣笛，短促，急躁。窗户没关严，
风从缝隙里挤进来，发出呜呜的声音，像
有人在哭。
```

### 嗅觉描写

**气味类型**：
- 自然气味（花香、泥土、雨水）
- 食物气味（饭菜、香料、腐败）
- 人造气味（香水、油漆、消毒水）
- 身体气味（汗味、体味、呼吸）
- 记忆气味（老房子、旧书、童年）

**描写技巧**：
- 用具体事物比喻（"像...的味道"）
- 关联情感记忆
- 描写气味的动态（飘散、弥漫、钻进）
- 描写生理反应（刺鼻、呛人、清新）

**示例**：
```
劣质版：房间里有股怪味。

优质版：
推开门，一股陈年的味道扑面而来——灰尘、
霉斑、还有某种说不清的有机物腐败的气息。
像是有人把夏天忘在角落里，任其发酵。
深吸一口，肺里都是湿漉漉的，能尝到
时间的味道。
```

### 味觉描写

**基本味觉**：
- 酸、甜、苦、辣、咸
- 鲜、涩、麻、凉、温

**复合描写**：
- 口感（脆、软、糯、硬、滑）
- 温度（烫、温、凉、冰）
- 余味（回甘、 lingering、消失快慢）

**示例**：
```
劣质版：咖啡很苦。

优质版：
第一口烫得舌尖发麻，随后是浓烈的苦，
像中药，像生活，像所有不愿意回忆的
往事。但苦味散去后，舌根泛起一丝若有
若无的酸，然后是焦香，像是有人把
黑夜烘焙后冲泡给你喝。
```

### 触觉描写

**触觉类型**：
- 温度（冷、热、温、凉）
- 质地（粗糙、光滑、柔软、坚硬）
- 压力（轻、重、挤压、拉伸）
- 疼痛（刺痛、钝痛、灼痛）
- 内部感觉（饥饿、疲劳、紧张）

**示例**：
```
劣质版：他的手很粗糙。

优质版：
握住他的手时，她吓了一跳。那不像人的手，
像老树皮，硬茧磨得她掌心发疼。指关节
粗大变形，指甲缝里嵌着洗不净的黑。
这双手挖过煤，搬过砖，打过铁，现在
握着她的，小心翼翼，像是怕弄碎什么
珍贵的东西。
```

### 五感综合练习

**场景**：雨天的咖啡馆

**五感描写清单**：
```
视觉：雨滴在玻璃上的轨迹、咖啡表面的油脂、
      顾客模糊的侧脸

听觉：雨声、咖啡机蒸汽声、低语、爵士乐

嗅觉：咖啡豆的焦香、雨水的清新、
      某人的香水味

味觉：咖啡的苦与酸、蛋糕的甜、
      舌尖的涩

触觉：杯子的温度、沙发的柔软、
      潮湿的空气贴在皮肤上
```

**完整示例**：
```
雨在玻璃上画出一道道歪斜的线，外面的世界
被扭曲成印象派画作。咖啡机发出满足的叹息，
蒸汽"嗤"地一声，像是完成了什么使命。
爵士乐从角落里流淌出来，萨克斯风懒洋洋的，
像只晒着太阳的猫。

她端起杯子，瓷的温热透过掌心。第一口，
咖啡的苦在舌尖炸开，随后是果酸，最后是
巧克力的回甘。芝士蛋糕在叉子下投降，
甜得罪恶。

邻桌的情侣低声说话，听不清内容，只看见
女孩笑的时候眼睛弯成月牙。空气里有雨水的
味道，混着咖啡香，还有她身上若有若无的
茉莉花香。

沙发把她整个人吞进去，柔软得像云。她
调整了一下姿势，羊毛毯摩擦着脸颊，有点痒。
窗外的雨下得更大了，但这里很暖，暖得
让人忘记时间。

### C3. 比喻和修辞使用指南

```markdown
## 比喻与修辞的恰当使用

### 比喻使用原则

**好比喻的标准**：
- ✅ 新颖但不生僻
- ✅ 贴切且易理解
- ✅ 能唤起情感或画面
- ✅ 符合语境和人物

**坏比喻的特征**：
- ❌ 陈词滥调（"如花似玉"、"心急如焚"）
- ❌ 过度复杂（需要解释才能懂）
- ❌ 情感错位（轻松场景用沉重比喻）
- ❌ 人物不符（古人说现代比喻）

### 比喻类型与示例

**明喻**（像、如、仿佛）：
```
陈词滥调：她美得像花。
优质比喻：她笑起来的时候，整条街的灯
          都亮了一度。

陈词滥调：他跑得很快。
优质比喻：他跑起来像身后有鬼在追，
          还是那种腿长的鬼。
```

**暗喻**（是、成为）：
```
陈词滥调：时间就是金钱。
优质比喻：时间是贼，偷走一切，
          连回忆都要收利息。

陈词滥调：他是只狐狸。
优质比喻：他是那种会在葬礼上
          推销保险的人。
```

**借喻**（用相关事物代指）：
```
陈词滥调：白大褂来了。（医生）
优质比喻：那顶乌纱帽压弯了多少脊梁。

陈词滥调：她穿上了红嫁衣。
优质比喻：她把青春缝进了那件婚纱里。
```

**通感**（感官互通）：
```
优质示例：
- 她的声音甜得发腻
- 那笑声是金色的
- 空气里弥漫着离别的味道
- 他的眼神冷得能冻伤人
```

### 其他修辞手法

**排比**（增强气势）：
```
弱：他很累，很困，很想放弃。
强：他累得睁不开眼，困得站不稳，
    累得想就这样躺下，再也不起来。
```

**对比**（突出差异）：
```
弱：以前很好，现在很糟。
强：三年前，这里满是孩子的笑声；
    现在，连风都不敢进来。
```

**反复**（强调情感）：
```
弱：他一直在等她。
强：等了一天，等了一年，等了一辈子。
    她没来，她没来，她始终没来。
```

**夸张**（渲染效果）：
```
弱：他很饿。
强：他饿得能吞下一头牛，连骨头都不吐。
```

**拟人**（赋予生命）：
```
弱：风很大。
强：风在窗外咆哮，像是要闯进来，
    告诉所有人它的不满。
```

### 修辞密度控制

```
单段落修辞建议：
- 关键段落：2-3 个修辞
- 一般段落：1-2 个修辞
- 过渡段落：0-1 个修辞

整章修辞分布：
- 开场：1-2 个（吸引注意）
- 发展：3-5 个（丰富描写）
- 高潮：2-3 个（不要喧宾夺主）
- 结尾：1-2 个（留余韵）
```

### 修辞自检清单

- [ ] 比喻是否新颖？
- [ ] 比喻是否贴切？
- [ ] 是否避免陈词滥调？
- [ ] 修辞是否服务于情感？
- [ ] 修辞密度是否合适？
- [ ] 是否符合人物和语境？

### 修辞修改练习

**原句**：她很伤心。

**逐步优化**：
```
版本 1（加比喻）：她的心像被刀割一样疼。
版本 2（换新颖比喻）：她的心像被掏空的
        房子，风从四面八方灌进来。
版本 3（加细节）：她坐在床边，手按在
        胸口，那里空得能听见回声。
版本 4（加动作）：她把自己蜷成一团，
        像是这样就能把心拼回去。
版本 5（最终版）：她坐在床边，手按在
        胸口。那里空了一块，风呼呼地
        往里灌。她试着蜷缩起来，像胎儿
        在子宫里的姿势，想着也许这样
        就能把心拼回去。但有些东西碎了，
        就是碎了。

### C4. 对话写作技巧

```markdown
## 对话写作：让人物"说话"

### 好对话的标准

- ✅ 符合人物身份和性格
- ✅ 有潜台词（话里有话）
- ✅ 推动情节或揭示人物
- ✅ 自然，像真人说话
- ✅ 有节奏和韵律

### 潜台词写作

**表面意思 vs 真实意思**：

| 表面台词 | 潜台词 | 情境 |
|----------|--------|------|
| "你忙吧。" | "你不在乎我。" | 恋人冷战 |
| "随便你。" | "我很在意，但我不想说。" | 赌气 |
| "挺好的。" | "糟透了，但我不想说。" | 寒暄 |
| "我没生气。" | "我非常生气。" | 明显生气时 |
| "你开心就好。" | "我不开心，但我不重要。" | 委屈 |

**潜台词示例**：
```
表面：
"你回来了。"她说。
"嗯。"他说。
"吃饭了吗？"
"吃了。"
"那就好。"

潜台词：
她想说"你为什么这么晚才回来"，
他想说"别管我"，她想说"我还爱你"，
他想说"我知道"。但最后，他们只说了
这些。有些话，说出口就变了味。
```

### 对话个性化

**不同人物的说话特征**：

| 人物类型 | 句式 | 词汇 | 节奏 | 习惯语 |
|----------|------|------|------|--------|
| 老人 | 长句，爱回忆 | 传统，方言 | 缓慢 | "想当年..." |
| 孩子 | 短句，重复 | 简单，具体 | 跳跃 | "然后...然后..." |
| 知识分子 | 复杂句，从句 | 抽象，专业 | 平稳 | "从某种意义上说..." |
| 工人 | 短句，直接 | 具体，粗粝 | 快 | "废话少说" |
| 商人 | 条件句，假设 | 利益相关 | 算计 | "你懂的" |

**个性化示例**：
```
同一意思，不同人说：

"我不同意。"

老人："这个嘛，恐怕不太妥当。想当年..."
孩子："不要不要！我不要这样！"
知识分子："从理论上讲，这个方案存在
           若干值得商榷的地方。"
工人："不行。没得商量。"
商人："这个...我们再考虑考虑成本问题。"
```

### 对话节奏控制

**紧张对话**（短句、打断）：
```
"你去了哪里？"
"出去。"
"和谁？"
"朋友。"
"什么朋友？"
"你管不着。"
"我是你妈！"
"那又怎样？"
```

**轻松对话**（长句、玩笑）：
```
"你说，要是我们老了，会是什么样子？"
"大概就是你满脸皱纹，我满嘴假牙，
然后还要互相嫌弃对方打呼噜吧。"
"真浪漫。"
"是吧，我也觉得。"
```

**情感对话**（停顿、省略）：
```
"我..."
"嗯？"
"没什么。"
"你说。"
"其实...算了。"
"我在听。"
"我爱你。"
"......我知道。"
```

### 对话标签技巧

**避免单调**：
```
单调版：
"你好。"他说。
"你好。"她说。
"最近怎么样？"他问。
"挺好的。"她答。

优化版：
"你好。"他伸出手。
她握了握，掌心微凉："你好。"
"最近怎么样？"他收回手，插进口袋。
她耸耸肩："挺好的。"
```

**用动作代替"说"**：
```
"我不去。"她把包摔在桌上。
"你必须去。"他盯着她，不眨眼。
"凭什么？"她迎上他的目光。
"就凭这个。"他推过一张纸。
她扫了一眼，脸色白了："你威胁我？"
"我只是陈述事实。"
```

### 对话信息传递

**避免"正如你所知"**：
```
❌ 坏例子：
"正如你所知，我们公司是三年前成立的，
主要生产电子产品，对吧？"
"是的，我知道。"

✅ 好例子：
"公司三年了，不容易。"老板摸着墙上的
营业执照，"还记得吗？那时候就我们俩，
在地下室吃泡面。"
"怎么不记得。"老张笑了笑，"你欠我
三个月工资，到现在还没给。"
"咳咳...今天天气不错。"
```

### 对话自检清单

- [ ] 每段对话是否有目的？
- [ ] 是否符合人物身份？
- [ ] 是否有潜台词？
- [ ] 是否自然流畅？
- [ ] 是否避免信息倾销？
- [ ] 是否有节奏变化？
```

### C5. 动作描写的层次感

```markdown
## 动作描写：从平庸到精彩

### 动作描写层次

**层次 1：基础动作**（做了什么）
```
他拿起杯子，喝了一口水。
```

**层次 2：加入细节**（怎么做）
```
他伸出右手，握住杯柄，举到唇边，
小口啜饮。
```

**层次 3：加入状态**（为什么这样做）
```
他的手有些抖，握住杯柄时用力过猛，
指节发白。举到唇边时停了一秒，
像是在犹豫什么，然后才小口啜饮。
```

**层次 4：加入情感**（内心感受）
```
他的手有些抖——三天没合眼了。握住
杯柄时用力过猛，指节发白，像是那是
救命稻草。举到唇边时停了一秒，
水面上映出他的脸，憔悴得陌生。
他想起上次这样喝水，还是在医院，
她躺在病床上，也这样看着他。
然后他才小口啜饮，水很凉，顺着
喉咙下去，浇不灭心里的火。
```

### 动作分解技巧

**将一个动作分解为多个微动作**：
```
原句：他点了一支烟。

分解：
他从口袋里摸出烟盒，空的。又摸，
摸出皱巴巴的一支，叼在嘴上。
打火机按了三下才着，火苗蹿起来，
照亮他眼里的血丝。烟头红了一瞬，
他深吸一口，闭眼，吐出的烟圈在
空气里散开，像是叹息。
```

### 动作 - 情感连接

**用动作表达情感**：
```
愤怒：他把手机砸在墙上，电池飞出来，
      滚到床底下。

悲伤：她慢慢叠着他的衣服，一件一件，
      叠得很整齐，像是他还会回来穿。

紧张：他不停看表，第五遍了。手指在
      腿上敲，敲出混乱的节奏。

喜悦：她跳起来，转了个圈，裙摆飞扬，
      像是整个人要飞起来。

绝望：他坐在床边，一动不动，像一尊
      被遗忘的雕塑。天黑了，他没开灯。
```

### 动作链设计

**连续动作的节奏感**：
```
快速动作链（紧张场景）：
他转身，拔腿，冲刺。楼梯在脚下，
一步两级。扶手擦过掌心，火辣辣的。
转角，撞上人，道歉，继续。肺在烧，
腿在抖，不能停。

慢速动作链（情感场景）：
她伸出手，在空中停了一秒，然后
缓缓落下，抚上他的脸。指尖触到
皮肤，微凉。拇指轻轻摩挲他的
颧骨，一下，两下，像是在确认
这是真的。然后她笑了，眼泪
掉下来，砸在他手背上。
```

### 动作描写禁忌

- ❌ 过度分解（每个动作都写，拖沓）
- ❌ 动作与情感脱节（为写而写）
- ❌ 违反物理规律（不可能的动作）
- ❌ 重复单调（总是同样的动作）
- ❌ 忽略环境互动（动作在真空中）

### 动作描写练习

**题目**：写"她离开"，不用"离开"、"走"等字

**示例**：
```
她拿起包，动作很轻，像是怕惊醒什么。
走到门口，手放在门把上，停住。一秒，
两秒，三秒。然后转动，"咔哒"一声，
门开了。她迈过门槛，没有回头。
门缓缓合上，把她的背影切成两半，
然后吞没。房间里只剩下空气，
还在微微震动。
