## 1. AI 写作核心原则

### 原则 1：展示而非讲述 (Show, Don't Tell)

**❌ 错误示例**：
```
小张很紧张。
```

**✅ 正确示例**：
```
小张的手指不自觉地捏着衣角，指节泛白。
她深吸一口气，声音还是微微发颤。
```

**原理**：
- 通过动作、表情、细节展示情绪
- 让读者自己感受，而非直接告知
- AI 容易直接"告知"，需要刻意练习"展示"

---

### 原则 2：情绪弧线连续

**❌ 错误示例**：
```
镜头 A：小张焦虑（紧张情绪）
镜头 B：小张感动（直接跳到感动，缺少过渡）
```

**✅ 正确示例**：
```
镜头 A：小张焦虑（紧张→愧疚）
镜头 B：小张意外（愧疚→意外）
镜头 C：小张感动（意外→感动）
```

**原理**：
- 情绪变化要渐进，不要跳跃
- 每个情绪转折点要有触发事件
- AI 容易忽略情绪过渡，需要明确指定

---

### 原则 3：镜头内变化优先

**❌ 错误示例**：
```
镜头 1：老李说话
镜头 2：小张反应
镜头 3：老李继续说话
镜头 4：小张再反应
```

**✅ 正确示例**：
```
镜头 1：老李说话，小张在背景中反应（5 秒）
镜头 2：小张说话，老李在背景中反应（5 秒）
```

**原理**：
- 能用一个镜头讲清楚，不要切镜头
- 减少渲染次数，降低成本
- 视觉更流畅，情绪不中断

---

### 原则 4：对话完整保留

**❌ 错误示例**：
```
原对话（20 字）："孩子半夜发烧，我带他去医院，刚回来"
AI 精简（8 字）："孩子发烧，刚回来"
```

**✅ 正确示例**：
```
原对话（20 字）："孩子半夜发烧，我带他去医院，刚回来"
保留完整，调整语速到 1.3 倍（约 5 秒）
```

**原理**：
- 核心对话承载情感和情节
- 删减会损失情感张力
- 用语速调整替代删减

---

### 原则 5：五感描写

**视觉**：
```
晨光透过玻璃窗，在老李的围裙上投下斑驳光影。
```

**听觉**：
```
蒸笼开盖的噗嗤声，碗碟碰撞的清脆声。
```

**嗅觉**：
```
豆浆的香气混合着面粉的味道，扑面而来。
```

**味觉**：
```
油条外酥里嫩，豆浆甜而不腻。
```

**触觉**：
```
热豆浆捧在手心，温暖从指尖传到心里。
```

**原理**：
- 五感描写增加画面感和代入感
- AI 容易只写视觉，需要刻意加入其他感官
- 每个场景至少使用 2-3 种感官

## 2. 提示词工程最佳实践

### 2.1 六要素公式

```
[主体与外貌细节] + [动作与物理连贯性] + [场景环境] +
[视觉风格/物理光影] + [物理焦段与运镜] + [原生音效要求]
```

**示例**：
```
45 岁中国男性，微胖圆脸，白色围裙沾着面粉，双手同时操作豆浆机和蒸笼，
动作娴熟但略显匆忙，额头渗出细密汗珠。
光影：店内顶灯暖黄 + 门口晨光斜射 + 蒸汽漫反射（光源层），
汗珠高光闪烁 + 面粉颗粒在光柱中漂浮 + 围裙褶皱阴影（光行为层），
暖黄主调 + 局部冷白高光（色调层）。
音效：豆浆机运转声、蒸笼开盖噗嗤声、碗碟碰撞声、老李喘息声。
```

---

### 2.2 光影三层结构

**光源层**：主光/辅光/背光位置
```
清晨侧逆光暖黄色 + 蒸笼白汽漫反射 + 老街散射暖光
```

**光行为层**：反射/折射/散射/体积光
```
晨雾柔化建筑轮廓 + 蒸汽体积光穿透 + 树影斑驳投影
```

**色调层**：冷暖对比/色彩公式
```
暖黄底调 + 青灰阴影冷暖对撞
```

**完整格式**：
```
光影：[光源层] + [光行为层] + [色调层]
```

---

### 2.3 对话时长计算

**核心公式**：
```
镜头时长 (秒) = 台词字数 / (语速 字/分钟 / 60)
```

**语速参考**：
| 语速 | 字/分钟 | 字/秒 | 适用场景 |
|------|---------|-------|----------|
| 正常 | 180 | 3.0 | 大部分对话 |
| 1.3 倍 | 234 | 3.9 | 紧张/急促对话 |
| 1.5 倍 | 270 | 4.5 | 快速对话 |
| 机械音 | 300 | 5.0 | 系统 VO |

**示例**：
```
台词："孩子半夜发烧，我带他去医院，刚回来"（18 字）
正常语速：18 / 3 = 6 秒
1.3 倍速：18 / 3.9 = 4.6 秒
```

## 3. 故事结构模板

### 3.1 三幕结构（90 秒短剧）

**第一幕：铺垫（0-30 秒）**
- 建立场景和角色
- 展示日常生活
- 激励事件（打破平衡）

**第二幕：对抗（30-70 秒）**
- 尝试解决问题
- 遇到阻碍和挫折
- 中点转折（从被动到主动）
- 危机逼近

**第三幕：解决（70-90 秒）**
- 高潮（最终对决）
- 结局（新的平衡）
- 主题升华

---

### 3.2 情绪弧线设计

**标准情绪弧线**：
```
平静 → 焦虑 → 意外 → 感动 → 温暖
  ↓       ↓       ↓       ↓       ↓
建立   冲突   转折   高潮   收尾
```

**示例（喜磨坊第 2 集）**：
```
镜头 1-3：平静/担忧（老李独自忙碌）
镜头 4-8：焦虑/紧张（小张迟到，孩子生病）
镜头 9-14：意外/感动（老李主动帮忙）
镜头 15-20：温暖/治愈（小宇吃早餐，找到家的感觉）
```

## 4. 角色发展指南

### 4.1 角色档案模板

```markdown
## 角色名：老李

**基本信息**：
- 年龄：45 岁
- 职业：早餐店老板
- 外貌：微胖圆脸，白色围裙沾面粉

**性格特征**：
- 核心：热心肠，嘴笨心热
- 表面：实在，不善言辞
- 深层：善良，有担当

**语言习惯**：
- 语速：中等（180 字/分钟）
- 口头禅："说啥说"、"这就够了"
- 用词：简短，直接，少修饰

**情绪表达**：
- 关心时：避开对视，用行动表达
- 生气时：声音低沉，动作加快
- 开心时：嘴角上扬，话变多

**角色弧线**：
- 起点：独自忙碌的小老板
- 转折：主动帮助小张
- 终点：社区大家长
```

---

### 4.2 角色关系图

```
        老李
       /    \
      /      \
   雇佣      常客
    /          \
   /            \
  小张 ———— 汪大妈
   |        (关怀)
   |
  小宇
```

**关系发展**：
- 老李 ↔ 小张：老板→家人
- 老李 ↔ 汪大妈：店主→老友
- 小张 ↔ 汪大妈：陌生→姐妹
- 小宇 ↔ 所有人：孩子→大家的宝贝

## 5. 对话写作技巧

### 5.1 潜台词设计

**表面台词**：
```
老李："三千五一个月，行的话明天来上班。四点半，别迟到。"
```

**潜台词**：
```
（我不多问你的困难，直接给你工作。工资合理，但你要守时。）
```

**原理**：
- 好对话有表面意思和深层意思
- 潜台词传达角色真实想法
- AI 容易写得太直白，需要刻意设计潜台词

---

### 5.2 对话节奏控制

**紧张场景**：
```
短句 + 快速交替 + 少量描述
```

**示例**：
```
"李哥，对不起我迟到了！"
"咋了这是？"
"孩子发烧，刚回来……"
"你过来。"
```

**温馨场景**：
```
长句 + 慢速交替 + 丰富描述
```

**示例**：
```
"好吃不？叔叔家豆浆，可好喝了。"老李摸摸小宇的头，眼神温和。
小宇嘴里塞满油条，用力点头："好喝！"
```

---

### 5.3 对话标注格式

**标准格式**：
```markdown
对白（角色，情绪）："台词内容"
```

**示例**：
```markdown
对白（老李，实在）："三千五一个月，行的话明天来上班。四点半，别迟到。"
对白（小张，意外）："啊？您不问问我……工资啥的？"
对白（小张，坚定）："一定准时到！"
```

**情绪标签库**：
```
焦急、愧疚、关切、认真、为难、决断、意外、实在、
温和、感动、轻声、感激、坚定、担忧、惊讶
```

## 6. 避免常见问题

### 6.1 流水账问题

**❌ 流水账**：
```
早上我起床了，然后去上班，中午吃饭，下午工作，晚上回家。
```

**✅ 优化后**：
```
晨光透过窗帘，我揉揉惺忪的睡眼，开始新的一天。
地铁里人潮涌动，我抓紧扶手，脑海里回放着待办清单。
午餐是楼下面馆的牛肉面，热气腾腾中短暂放空。
下午的会议室里，键盘敲击声和讨论声此起彼伏。
推开家门时，夜色已深，但桌上的台灯依旧温暖。
```

**避免方法**：
1. 场景选择原则（什么值得写，什么可以跳过）
2. 五感描写（至少 2-3 种感官）
3. 细节具体化（时间/地点/动作/感受）
4. 情绪贯穿（每个场景有情绪基调）

---

### 6.2 陈词滥调

**❌ 陈词滥调**：
```
与时俱进、再创辉煌、团结一心、努力奋斗
```

**✅ 具体描述**：
```
凌晨四点的街道还沉浸在睡梦中，老李已经打开了蒸笼。
白汽袅袅升起，在晨光中画出温暖的弧线。
```

**避免方法**：
1. 用具体细节替代抽象词汇
2. 用动作展示替代形容词
3. 用独特比喻替代常见表达

---

### 6.3 信息倾倒

**❌ 信息倾倒**：
```
小张，38 岁，单亲妈妈，独自带一个 6 岁的儿子，
之前在老家帮过早餐店，新搬来这个小区，
现在急需一份工作养活孩子……
```

**✅ 信息融入情节**：
```
"新搬来的，住后面那栋。看您忙不过来，搭把手。"
小张熟练地装袋、收钱、招呼客人，动作利落。
"以前在老家帮过早餐店。"她笑着说。
```

**避免方法**：
1. 信息通过对话自然透露
2. 信息通过动作展示
3. 信息分多次透露，不要一次性

## 7. AI 视频脚本优化

### 7.1 镜头衔接设计

**7 种衔接技巧**：

| 技巧 | 说明 | 示例 |
|------|------|------|
| **动作匹配** | 上镜头动作结束=下镜头动作开始 | 伸手递物→伸手接物 |
| **视线匹配** | 上镜头视线方向=下镜头人物位置 | A 看右→B 在右 |
| **光影匹配** | 相邻镜头光影方向一致 | 侧光→侧光 |
| **色调匹配** | 相邻镜头色调渐变合理 | 暖黄→暖黄 |
| **声音桥接** | 上镜头声音延续到下镜头 | 说话声→环境音 |
| **焦点转换** | 焦点从 A 转移到 B | 前景实→背景实 |
| **运动匹配** | 运动方向/速度一致 | 左→右跟拍 |

---

### 7.2 视觉化描写

**❌ 不适合 AI 视频**：
```
他心里想着……（内心活动无法可视化）
```

**✅ 适合 AI 视频**：
```
他眉头紧锁，手指轻轻敲击桌面。（动作可可视化）
```

**原理**：
- AI 视频只能生成可见内容
- 内心活动要通过动作/表情展示
- 抽象概念要转化为具体视觉元素

---

### 7.3 提示词长度控制

**Seedance(即梦) 平台**：
- 中文提示词 ≤ 500 字符
- 单个镜头 ≤ 15 秒
- 超过 15 秒需分段拼接

**优化技巧**：
1. 删除冗余词（"杰作"、"4k"、"超清晰"）
2. 用物理材质词替代质量词
3. 运镜词用中文（推轨推进，不用 Dolly）

## 8. 实战案例库

### 8.1 喜磨坊第 1 集《新来的》

**核心场景**：小张面试

**情绪弧线**：
```
紧张（面试）→ 意外（被录用）→ 感动（送早餐）
```

**关键对话**：
```
老李："带孩子的妈，能吃苦。这就够了。"
（潜台词：我理解你的困难，直接给你工作）
```

**镜头设计**：
- 镜头 13：老李装早餐（6 秒）
- 镜头 14：小张愣住（4 秒）
- 镜头 15：老李塞早餐（5 秒）
- 衔接：动作匹配（递→接）、视线匹配（避开→看向）

---

### 8.2 喜磨坊第 2 集《迟到的早餐》

**核心场景**：老李接孩子

**情绪弧线**：
```
焦虑（迟到）→ 意外（老李帮忙）→ 感动（找到家）
```

**关键改进**：
- 原台词："放邻居家了，人家说只能帮一上午"
- 优化后："放邻居家了，那安静，能好好休息"
- 改进理由：强调孩子能更好休息，减少愧疚感
