# 写作风格指南 - Writing Style Guide

## 一、语气与风格分类

### 1. 正式风格 (Formal)

**适用场景**：商务邮件、学术报告、官方文件、法律文档

**特征**：
- 使用完整句式，避免缩写
- 第三人称视角为主
- 专业术语准确使用
- 客观、中立、克制
- 避免口语化表达

**示例**：
> ❌ "我们觉得这个方案挺好的"
> ✅ "经评估，该方案具有显著优势"

**常用词汇**：
- 因此、然而、此外、鉴于、综上所述
- 建议、推荐、评估、分析、确认

---

### 2. 休闲风格 (Casual)

**适用场景**：社交媒体、朋友聊天、博客文章、个人邮件

**特征**：
- 第一/第二人称视角
- 口语化、亲切自然
- 可使用缩写和网络用语
- 情感表达直接
- 句式灵活多变

**示例**：
> ❌ "本人对此事持有不同意见"
> ✅ "我有点不太同意这个看法"

**常用词汇**：
- 其实、真的、超级、特别、搞定
- 哈、呢、吧、啦（语气词适度使用）

---

### 3. 专业风格 (Professional)

**适用场景**：行业报告、技术文档、项目方案、产品介绍

**特征**：
- 术语准确、表达精准
- 逻辑清晰、层次分明
- 数据支撑论点
- 客观但不失温度
- 避免过度修饰

**示例**：
> ❌ "这个产品超级好用，大家都爱"
> ✅ "该产品用户满意度达 92%，复购率提升至 78%"

**常用词汇**：
- 基于、针对、实现、优化、提升
- 数据、指标、效率、转化率、ROI

---

### 4. 创意风格 (Creative)

**适用场景**：故事、诗歌、广告文案、品牌内容

**特征**：
- 意象丰富、画面感强
- 修辞手法多样
- 节奏感和韵律感
- 情感共鸣优先
- 突破常规表达

**示例**：
> ❌ "太阳下山了，天黑了"
> ✅ "夕阳收起最后一抹金辉，夜幕悄然降临"

**常用手法**：
- 比喻、拟人、排比、对比
- 感官描写（视觉、听觉、嗅觉、触觉）
- 留白与暗示

---

### 5. 说服风格 (Persuasive)

**适用场景**：销售文案、演讲稿、议论文、提案

**特征**：
- 明确的立场和观点
- 强有力的论据支撑
- 情感与理性结合
- 清晰的行动号召
- 预判并回应异议

**结构框架**：
```
1. 吸引注意（钩子）
2. 阐述问题（痛点）
3. 提供方案（解决）
4. 证明有效（证据）
5. 行动号召（CTA）
```

## 二、文体写作规范

### 📄 文章写作

**开头**：
- 开门见山：直接点明主题
- 设问引入：引发读者思考
- 故事开场：建立情感连接
- 数据冲击：用事实吸引注意

**正文**：
- 段落分明，每段一个核心观点
- 使用过渡句连接段落
- 论据充分，逻辑严密
- 适当使用小标题分层

**结尾**：
- 总结要点，强化印象
- 升华主题，引发思考
- 行动号召，明确下一步
- 首尾呼应，结构完整

---

### 📧 邮件写作

**主题行**：
- 简洁明了（≤50 字符）
- 包含关键信息
- 避免模糊词汇

**正文结构**：
```
称呼 → 开场白 → 核心内容 → 行动要求 → 结束语 → 签名
```

**语气要点**：
- 尊重但不卑微
- 清晰但不生硬
- 简洁但不失礼貌

**常用模板**：
```
尊敬的 [姓名]：

您好！[开场寒暄/背景说明]

[核心内容，分点陈述]

[期望的行动/回复]

感谢您的时间！

此致
[你的名字]
```

---

### 📊 报告写作

**标准结构**：
1. 执行摘要（1 页内概括核心内容）
2. 背景与目的
3. 方法与过程
4. 结果与分析
5. 结论与建议
6. 附录（数据、图表、参考）

**写作要点**：
- 用数据说话
- 图表辅助说明
- 结论先行，细节在后
- 建议具体可执行

---

### 📖 故事写作

**核心要素**：
- **人物**：立体、有动机、有成长
- **冲突**：内在/外在、人与环境
- **情节**：起承转合、节奏把控
- **场景**：具体、可感、服务于主题
- **主题**：贯穿始终的核心思想

**叙事技巧**：
- 展示而非讲述（Show, Don't Tell）
- 对话推动情节
- 细节塑造真实感
- 悬念保持吸引力

## 三、常见写作错误与避免方法

### ❌ 错误类型及修正

| 错误类型 | 问题示例 | 修正建议 |
|----------|----------|----------|
| **冗词赘句** | "在这个情况下，我们不得不说..." | "此时我们必须..." |
| **被动语态过度** | "这个决定被做出了" | "我们做出了这个决定" |
| **模糊表达** | "大概、可能、也许" | 给出具体数据或明确判断 |
| **逻辑跳跃** | 缺少过渡，前后不连贯 | 添加连接词和过渡句 |
| **重复啰嗦** | 同一意思反复说 | 删减冗余，保留精华 |
| **语气不一致** | 正式与口语混用 | 全文保持统一风格 |
| **长句难懂** | 一句话超过 50 字 | 拆分长句，简化结构 |
| **陈词滥调** | "与时俱进"、"再创辉煌" | 用具体描述替代套话 |

---

### ✅ 自查清单

写完内容后，问自己：

- [ ] 核心观点是否清晰？
- [ ] 逻辑是否连贯？
- [ ] 用词是否准确？
- [ ] 语气是否合适？
- [ ] 有无冗余内容？
- [ ] 格式是否规范？
- [ ] 有无语法错误？
- [ ] 读者能否理解？

## 四、润色与编辑流程

### 第一步：宏观审视（结构层）
1. 整体结构是否合理？
2. 段落顺序是否最优？
3. 有无缺失或冗余部分？
4. 主题是否贯穿始终？

### 第二步：中观调整（段落层）
1. 每段是否有明确主题句？
2. 段落间过渡是否自然？
3. 论据是否充分支撑论点？
4. 段落长度是否适中？

### 第三步：微观打磨（句子层）
1. 句子是否简洁有力？
2. 用词是否精准恰当？
3. 有无语法和拼写错误？
4. 标点使用是否规范？

### 第四步：风格统一（语气层）
1. 全文语气是否一致？
2. 人称使用是否统一？
3. 术语使用是否规范？
4. 格式排版是否整洁？

---

## 五、特殊场景写作提示

### 🌐 跨文化写作
- 避免文化特定隐喻
- 注意日期、数字格式
- 考虑翻译友好性
- 尊重文化差异

### 📱 移动端写作
- 短段落（3-4 行内）
- 小标题频繁使用
- 关键信息前置
- 避免复杂表格

### 🔍 SEO 写作
- 关键词自然融入
- 标题包含核心词
-  meta 描述简洁有力
- 内部链接合理布局
