## 🎯 各文档用途

### 1. CONSISTENCY.md - 一致性管理规则

**用途**：避免逻辑错误、设定遗忘、上下文断裂

**核心内容**：
- **A. 设定追踪系统**：人物/场景/情节/世界观档案模板
- **B. 上下文管理流程**：写作前/中/后全流程管理
- **C. 逻辑错误检测清单**：7 大检查项（含新增流水账检测）
  - C1. 时间线一致性检查
  - C2. 因果关系验证
  - C3. 人物行为合理性检查
  - C4. 场景转换逻辑检查
  - C5. 信息前后一致性检查
  - **C6. 流水账检测清单** ⭐ 新增
  - **C7. 文笔质量评估清单** ⭐ 新增
- **D. 记忆辅助工具**：摘要/伏笔追踪/修改记录
- **E. AI 写作专用提示词**：系统指令/上下文注入

**使用时机**：
- 开始写作前：建立设定档案
- 写作过程中：上下文管理
- 章节完成后：一致性审查

---

### 2. CHECKLIST.md - 写作检查清单

**用途**：每章完成后的系统性自查工具

**核心内容**：
- 📋 章节完成自查清单（基础检查）
- 👤 人物一致性检查项（外貌/性格/状态）
- 📖 情节逻辑检查项（因果/时间线/伏笔/冲突）
- 🌍 场景连贯性检查项（描述/匹配）
- ✍️ 语言风格一致性检查项（视角/语气/用词/节奏）
- 📊 综合评分表
- 🔧 快速修复指南

**使用时机**：
- 每章写作完成后
- 提交审阅前
- 定期质量检查

**使用方法**：
1. 打印或打开电子文档
2. 逐项勾选检查
3. 记录问题和修改方案
4. 评分并归档

---

### 3. STYLE_IMPROVEMENT.md - 文笔提升指南

**用途**：解决"流水账"问题，提升文学性

**核心内容**：
- **A. 流水账问题诊断**
  - 特征识别
  - AI 易犯原因
  - 与好故事的区别

- **B. 避免流水账的方法**
  - 场景选择原则（什么值得写）
  - 详略得当技巧
  - 节奏控制方法
  - 冲突与张力设计

- **C. 文笔提升技巧**
  - 展示而非讲述（Show, Don't Tell）
  - 五感描写训练
  - 比喻和修辞指南
  - 对话写作技巧
  - 动作描写层次

- **D. AI 写作专用优化提示词**
  - 避免流水账系统指令
  - 文笔润色指令模板
  - 场景深化指令
  - 情感渲染指令

- **E. 实战对比示例**
  - 流水账 vs 优化版本对照
  - 修改说明
  - 改写练习模板

- **F. 文笔自检清单**
  - 画面感检查
  - 情感共鸣检查
  - 对话自然度检查
  - 描写精准度检查
  - 节奏合理性检查
  - 综合文笔评分表

**使用时机**：
- 感觉文笔不佳时
- 被指出"流水账"时
- 想要提升文学性时
- AI 写作优化时

## 🔄 完整写作流程

### 阶段 1：写作准备

```
1. 阅读 CONSISTENCY.md
   ├─→ 创建人物档案（A1）
   ├─→ 创建场景档案（A2）
   ├─→ 创建情节档案（A3）
   └─→ 创建世界观档案（A4）

2. 使用上下文管理流程（B）
   ├─→ 写作前设定回顾清单（B1）
   ├─→ 准备上下文注入（E2）
   └─→ 设定本章目标
```

### 阶段 2：写作过程

```
1. 使用 AI 提示词（E）
   ├─→ 系统提示词模板（E1）
   ├─→ 上下文注入（E2）
   └─→ 避免流水账指令（STYLE_IMPROVEMENT.md D1）

2. 实时检查（B2）
   ├─→ 每 500 字快速自检
   ├─→ 场景转换标记
   └─→ 设定约束确认

3. 文笔控制
   ├─→ 详略得当（B2）
   ├─→ 节奏控制（B3）
   └─→ 冲突设计（B4）
```

### 阶段 3：写作后审查

```
1. 一致性审查（CONSISTENCY.md C）
   ├─→ 时间线检查（C1）
   ├─→ 因果检查（C2）
   ├─→ 人物检查（C3）
   ├─→ 场景检查（C4）
   ├─→ 信息检查（C5）
   ├─→ 流水账检测（C6）⭐ 新增
   └─→ 文笔质量评估（C7）⭐ 新增

2. 清单自查（CHECKLIST.md）
   ├─→ 基础检查
   ├─→ 人物一致性
   ├─→ 情节逻辑
   ├─→ 场景连贯
   ├─→ 语言风格
   └─→ 综合评分

3. 文笔优化（STYLE_IMPROVEMENT.md）
   ├─→ 流水账修改（B）
   ├─→ 文笔提升技巧（C）
   ├─→ 使用润色指令（D）
   └─→ 对比示例参考（E）

4. 记录与更新（CONSISTENCY.md D）
   ├─→ 更新章节摘要（D2）
   ├─→ 更新伏笔追踪（D3）
   ├─→ 记录修改日志（D4）
   └─→ 更新设定档案
```

## 🎯 常见问题快速解决

### 问题 1：逻辑错误/前后矛盾

**使用文档**：CONSISTENCY.md

**解决步骤**：
1. 使用 C1-C5 检查清单找出问题
2. 查阅人物/场景/情节档案确认设定
3. 使用 D4 修改记录日志记录变更
4. 同步更新相关档案

---

### 问题 2：被说"流水账"

**使用文档**：STYLE_IMPROVEMENT.md

**解决步骤**：
1. 阅读 A 部分，诊断流水账特征
2. 使用 C6 流水账检测清单（CONSISTENCY.md）评估
3. 应用 B 部分方法（场景选择/详略/节奏）
4. 使用 D 部分 AI 指令优化
5. 参考 E 部分对比示例修改

---

### 问题 3：文笔不佳/缺乏文学性

**使用文档**：STYLE_IMPROVEMENT.md

**解决步骤**：
1. 使用 F 部分文笔自检清单评估
2. 学习 C 部分技巧（展示/五感/比喻/对话/动作）
3. 使用 D 部分 AI 指令润色
4. 练习 E 部分改写练习
5. 重复直到达标

---

### 问题 4：忘记设定/人物 OOC

**使用文档**：CONSISTENCY.md + CHECKLIST.md

**解决步骤**：
1. 查阅人物档案（A1）确认设定
2. 使用 CHECKLIST.md 人物一致性检查项
3. 修正 OOC 行为
4. 更新档案（如有意发展）

---

### 问题 5：章节质量不稳定

**使用文档**：CHECKLIST.md

**解决步骤**：
1. 每章使用综合评分表评分
2. 记录分数，找出弱项
3. 针对性改进（参考 STYLE_IMPROVEMENT.md）
4. 建立质量标准（目标：≥30 分）

## 🛠️ AI 提示词快速使用

### 避免流水账
```
【写作风格指令】

你是一位专业小说家，擅长生动细腻的叙事。
请遵循以下原则：
1. 拒绝流水账（不要平铺直叙）
2. 展示而非讲述（用细节展示情感）
3. 详略得当（重要场景详写，过渡略写）
4. 节奏变化（紧张用短句，舒缓用长句）
5. 文学性要求（新颖比喻、五感描写）

请创作：[具体内容]
```

### 文笔润色
```
【润色指令】

请对以下文字进行文学性润色：

原文：[粘贴]

要求：
1. 保持原意不变
2. 增强画面感和情感
3. 加入五感描写
4. 优化比喻修辞
5. 调整节奏
6. 对话增加潜台词

请提供润色版本和改动说明。
```

### 场景深化
```
【场景深化指令】

请将以下场景从"讲述"升级为"展示"：

原始概述：[1-2 句描述]

深化要求：
1. 场景设定（时间、地点、环境）
2. 人物呈现（外貌、动作、内心）
3. 冲突设计（张力、情绪变化）
4. 感官丰富（五感描写）
5. 文学技巧（比喻、象征、节奏）

请输出 1500-2000 字深化版本。
```
