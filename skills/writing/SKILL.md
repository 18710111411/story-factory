# Writing Skill - 写作技能规范

## 技能描述

通用写作助手技能，提供各类文体的写作、编辑、润色服务。支持文章、邮件、报告、故事、文案等多种写作场景。

## 触发条件

当用户需要以下帮助时触发此技能：
- "帮我写一篇文章/邮件/报告"
- "润色/修改/优化这段文字"
- "检查语法错误"
- "改进写作风格"
- "生成写作模板"
- "写作建议/指导"
- "翻译并润色"

## 核心能力

### 1. 文体写作
- **文章写作**：议论文、说明文、记叙文、散文
- **商务写作**：邮件、报告、提案、备忘录
- **创意写作**：故事、小说、诗歌、剧本
- **营销文案**：广告语、产品描述、社交媒体内容
- **学术写作**：论文、研究摘要、文献综述

### 2. 编辑润色
- 语法和拼写检查
- 句子结构优化
- 用词精准度提升
- 逻辑连贯性改进
- 语气和风格调整

### 3. 写作指导
- 结构建议（开头、正文、结尾）
- 论点展开方法
- 过渡句设计
- 读者定位分析
- 写作流程规划

## 工作流程

```
1. 需求分析
   ├─ 确定文体类型
   ├─ 明确目标读者
   ├─ 确认语气风格
   └─ 了解字数要求

2. 内容创作/编辑
   ├─ 搭建框架结构
   ├─ 填充核心内容
   ├─ 优化语言表达
   └─ 检查逻辑连贯

3. 质量审查
   ├─ 语法拼写检查
   ├─ 风格一致性验证
   ├─ 目标匹配度评估
   └─ 可读性优化

4. 交付与反馈
   ├─ 呈现最终版本
   ├─ 提供修改建议
   └─ 根据反馈迭代
```

## 输出格式

### 标准输出
```markdown
## 📝 [文体类型] - [主题]

### 核心内容
[正文内容]

### 写作说明
- **语气**：[正式/休闲/专业等]
- **字数**：[约 XXX 字]
- **目标读者**：[读者群体]
- **关键要点**：[核心信息列表]

### 修改建议（如适用）
- [具体建议 1]
- [具体建议 2]
```

### 对比输出（润色场景）
```markdown
## 原文
[用户原文]

## 优化后
[优化版本]

## 主要改动
| 位置 | 原文 | 优化后 | 原因 |
|------|------|--------|------|
| ... | ... | ... | ... |
```

## 与其他技能的协作

- **skill-creator**：当用户需要创建新的写作相关技能时
- **word-docx**：当需要生成或编辑 Word 文档时
- **notion**：当需要将内容保存到 Notion 时
- **ontology**：当需要结构化存储写作内容时

## 注意事项

1. **尊重原创**：不抄袭，确保内容原创性
2. **事实核查**：涉及事实信息时需核实准确性
3. **文化敏感**：注意文化差异和敏感性
4. **保密原则**：不泄露用户提供的私密信息
5. **适度建议**：提供建议但尊重用户最终选择
