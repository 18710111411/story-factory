# 镜界 AI LensCraft AI - 更新日志

## v1.1 更新 (2026-04-02)

### 🎯 主要更新

#### 1. 对话嵌入功能 ⭐

**更新前**：
- 对话与提示词分离
- 需要额外标注对话内容
- 对话时长需手动计算

**更新后**：
- 对话直接嵌入提示词
- 格式统一：`对白（角色，情绪）："台词内容"`
- 自动计算对话时长匹配

**示例**：
```markdown
**对话**：老李："三千五一个月，行的话明天来上班。四点半，别迟到。"

**提示词**：
```
...
对白（老李，实在）："三千五一个月，行的话明天来上班。四点半，别迟到。"
...
```
```

#### 2. 镜头衔接设计 ⭐

**新增规则**：7 种衔接技巧

| 技巧 | 说明 | 应用场景 |
|------|------|----------|
| **动作匹配** | 上镜头动作结束=下镜头动作开始 | 递物/接物/起身 |
| **视线匹配** | 上镜头视线方向=下镜头人物位置 | 对话/对视 |
| **光影匹配** | 相邻镜头光影方向一致 | 所有场景 |
| **色调匹配** | 相邻镜头色调渐变合理 | 情绪过渡 |
| **声音桥接** | 上镜头声音延续到下镜头 | 对话/环境音 |
| **焦点转换** | 焦点从 A 转移到 B | 注意力引导 |
| **运动匹配** | 运动方向/速度一致 | 跟拍/移动 |

**衔接注释格式**：
```markdown
## 镜头 X | [镜头名称]

**上接镜头 X-1**：
- 动作衔接：[上镜头动作结束] → [本镜头动作开始]
- 视线衔接：[上镜头视线方向] → [本镜头人物位置]
- 光影衔接：[上镜头光影] → [本镜头光影]
- 色调衔接：[上镜头色调] → [本镜头色调]

**下接镜头 X+1**：
- 动作衔接：[本镜头动作结束] → [下镜头动作开始]
- 视线衔接：[本镜头视线方向] → [下镜头人物位置]
```

### 🎯 使用建议

1. **对话设计**：
   - 每条对话标注角色和情绪
   - 对话时长控制在镜头时长的 70% 内
   - 关键台词预留充足时间

2. **衔接设计**：
   - 优先使用动作匹配和视线匹配
   - 光影/色调保持一致性
   - 情绪过渡要自然渐变

3. **检查流程**：
   - 生成前检查衔接设计
   - 生成后预览拼接效果
   - 调整不流畅的镜头
