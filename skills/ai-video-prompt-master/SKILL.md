## 🎯 核心能力

### 1. 剧本→分镜→提示词转换
- 分析剧本场景和情感
- 分解为镜头序列
- 生成优化的 AI 视频提示词

### 2. 导演思维注入
- 镜头内变化优先于切镜头
- 对话完整保留（对话嵌入提示词）
- CG 动画优势最大化
- 情绪弧线连续
- 镜头衔接流畅性设计
- **过肩 45 度镜头**：两人对话时使用过肩镜头增强空间感 ⭐ 新增

### 3. 多平台提示词优化
- Seedance(即梦)：15 秒规则，中文提示词≤500 字符
- Kling：复杂动作优化
- Veo3：高质量美学，长镜头
- Luma：快节奏优化

### 4. 光影三层结构
- **光源层**：主光/辅光/背光位置
- **光行为层**：反射/折射/散射/体积光
- **色调层**：冷暖对比/色彩公式

### Step 3: 镜头分解
**分解规则**：

#### 镜头密度标准
| 内容类型 | 镜头数/分钟 | 平均镜头时长 |
|----------|-------------|--------------|
| 压抑铺垫 | 15-18 | 3.3-4s |
| 对话冲突 | 18-22 | 2.7-3.3s |
| 动作高潮 | 22-28 | 2.1-2.7s |
| 情绪收尾 | 15-18 | 3.3-4s |

#### 镜头分类
| 类型 | 缩写 | 时长建议 | 用途 |
|------|------|----------|------|
| 大特写 | ECU | 2-4s | 眼睛/手部/细节 |
| 特写 | CU | 3-6s | 脸部表情 |
| 中近景 | MCS | 3-5s | 上半身 + 部分环境 |
| 中景 | MS | 4-6s | 全身 + 环境 |
| 全景 | FS | 3-5s | 环境全貌 |
| 大远景 | EWS | 3-6s | 建立场景 |

#### 镜头运动规范

| 运动 | 速度 | 用途 | 情感效果 |
|------|------|------|----------|
| **Push In** (缓推) | 3-5s | 强调情绪/细节 | 紧张、专注、内心波动 |
| **Pull Out** (缓拉) | 3-6s | 揭示环境/关系 | 孤独、疏离、全局视角 |
| **Rack Focus** (焦点转换) | 1-3s | 注意力转移 | 发现、觉醒、关系转变 |
| **Tracking** (跟随) | 跟随速度 | 角色移动 | 陪伴、紧张、行动感 |
| **环绕** (Orbital) | 4-8s | 气场/转折 | 震撼、觉醒、力量感 |
| **手持晃动** | 持续 | 真实感/紧张 | 不安、慌乱、纪实感 |
| **Dolly Zoom** (希区柯克变焦) | 2-4s | 心理冲击 | 震惊、恐惧、认知颠覆 |
| **Whip Pan** (甩镜头) | 0.5-1s | 快速转场 | 紧张、突发、时间跳跃 |
| **Crane Up/Down** (升降) | 3-6s | 视角转换 | 命运感、全局观、收束 |
| **Low Angle** (仰拍) | 静态 | 强调权威/力量 | 敬畏、压迫、崇高 |
| **High Angle** (俯拍) | 静态 | 强调弱小/无助 | 同情、脆弱、全局 |
| **Dutch Angle** (荷兰角) | 静态 | 心理失衡 | 不安、混乱、紧张 |
| **Extreme Close-Up** (极致特写) | 2-4s | 细节强调 | 紧张、 intimate、微观 |
| **Wide Shot** (广角) | 3-6s | 环境展示 | 孤独、开阔、氛围 |

**镜头语言选择原则**：
1. **情绪驱动**：根据角色情绪选择镜头运动
2. **节奏变化**：快慢镜头交替，避免单调
3. **视线引导**：用镜头引导观众注意力
4. **空间建立**：用广角/全景建立空间感
5. **细节强调**：用特写强调关键细节

**输出**：镜头分解表

### Step 4: 提示词生成
**六要素公式**：
```
[主体与外貌细节] + [动作与物理连贯性] + [场景环境] +
[视觉风格/物理光影] + [物理焦段与运镜] + [原生音效要求]
```

**时间戳分镜法**（>5 秒镜头）：
```
[风格/色调总纲]。
0-3 秒：[画面 + 镜头]。
3-8 秒：[画面 + 镜头]。
8-12 秒：[画面 + 镜头]。
光影：[光源层] + [光行为层] + [色调层]。
音效：[物理拟声描述]。
禁止：任何文字、字幕、LOGO 或水印
```

**对话时长计算**：
```
正常语速：180 字/分钟 = 3 字/秒
1.3 倍速：234 字/分钟 = 3.9 字/秒
1.5 倍速：270 字/分钟 = 4.5 字/秒
机械音：300 字/分钟 = 5 字/秒

镜头时长 (秒) = 台词字数 / (语速 字/分钟 / 60)
```

**对话嵌入格式**：
```
对白（角色，情绪）："台词内容"
画外音（角色，内心）："独白内容"
```

**输出**：原始提示词

### Step 4.5: 镜头衔接设计（新增）

**衔接原则**：
1. **视觉连续性**：相邻镜头保持光影/色调一致
2. **动作连续性**：动作起止点匹配（如伸手→接物）
3. **视线连续性**：视线方向匹配（如 A 看右→B 在右）
4. **情绪连续性**：情绪渐变不突兀（如焦虑→意外→感动）

**衔接技巧**：

| 技巧 | 说明 | 示例 |
|------|------|------|
| **动作匹配** | 上镜头动作结束=下镜头动作开始 | 伸手递物→伸手接物 |
| **视线匹配** | 上镜头视线方向=下镜头人物位置 | A 看右→B 在右侧入画 |
| **光影匹配** | 相邻镜头光影方向一致 | 侧光→侧光（非侧光→逆光） |
| **色调匹配** | 相邻镜头色调渐变 | 暖黄→暖黄（非暖黄→冷蓝） |
| **声音桥接** | 上镜头声音延续到下镜头 | 说话声→环境音（声音渐隐） |
| **焦点转换** | 焦点从 A 转移到 B | 前景虚→背景实 |
| **运动匹配** | 运动方向/速度一致 | 左→右 跟拍→左→右 跟拍 |

**衔接检查清单**：
```
□ 相邻镜头光影方向一致
□ 相邻镜头色调渐变合理
□ 动作起止点匹配
□ 视线方向匹配
□ 情绪过渡自然
□ 声音设计连贯
□ 无跳跃式转场（除非刻意设计）
```

**输出**：镜头衔接报告

### Step 5: 质量校验
**检查清单**：

```
□ 每个镜头的对话字数在时长限制内

□ 关键台词有足够时长 (非匆忙带过)
□ 情绪台词有停顿空间 (非填满)
□ 画外音/重叠处理合理
□ 总对话时长 ≤ 总时长的 70%
```

#### 前期检查
```
□ 镜头密度合理 (18-22 镜头/分钟)
□ 对话完整保留 (语速/重叠优化)
□ 镜头内变化最大化
□ CG 优势充分利用
□ 情绪弧线连续
```

#### 镜头检查

```
□ 每个镜头有内部变化 (非静态)
□ 焦点/运动/表演至少一种变化
□ 时长与情绪匹配
□ 灯光符合情绪
□ 构图符合 16:9 优化
```

#### 衔接检查（新增）
```
□ 相邻镜头光影方向一致
□ 相邻镜头色调渐变合理
□ 动作起止点匹配
□ 视线方向匹配
□ 情绪过渡自然
□ 声音设计连贯
□ 无跳跃式转场（除非刻意设计）
```

```
□ 音效描述独立
□ 禁止项标准
```

## 🎬 镜头衔接设计规则（新增）

### 衔接原则

| 原则 | 说明 | 检查项 |
|------|------|--------|
| **视觉连续性** | 相邻镜头保持光影/色调一致 | 光影方向/色调渐变 |
| **动作连续性** | 动作起止点匹配 | 伸手→接物/起身→站立 |
| **视线连续性** | 视线方向匹配 | A 看右→B 在右/A 低头→B 在高处 |
| **情绪连续性** | 情绪渐变不突兀 | 焦虑→意外→感动（非焦虑→狂喜） |

---

### 衔接技巧详解

#### 1. 动作匹配剪辑

**原理**：上镜头动作结束=下镜头动作开始

**示例**：
```
镜头 A（结尾）：老李伸手递出早餐袋，手臂伸到画面边缘
镜头 B（开始）：小张的手从画面边缘伸入，接住早餐袋
```

**提示词设计**：
```
镜头 A：...老李伸手递出早餐袋，手臂伸向画面右侧边缘...
镜头 B：...小张的手从画面左侧伸入，接住早餐袋...
```

---

#### 2. 视线匹配剪辑

**原理**：上镜头视线方向=下镜头人物位置

**示例**：
```
镜头 A（结尾）：老李看向画面右侧，眼神关切
镜头 B（开始）：小张从画面右侧入画，站在老李对面
```

**提示词设计**：
```
镜头 A：...老李转头看向画面右侧，眼神关切...
镜头 B：...小张从画面右侧入画，站在老李对面...
```

---

#### 3. 光影匹配剪辑

**原理**：相邻镜头光影方向一致

**示例**：
```
镜头 A：侧光暖黄，人物面部左亮右暗
镜头 B：侧光暖黄，人物面部左亮右暗（非逆光/顶光）
```

**提示词设计**：
```
镜头 A：光影：侧光暖黄 + 面部柔光 + ...
镜头 B：光影：侧光暖黄 + 面部柔光 + ...
```

---

#### 4. 色调匹配剪辑

**原理**：相邻镜头色调渐变合理

**示例**：
```
镜头 A：暖黄底调 + 浅灰衣服中性调
镜头 B：暖黄底调 + 深棕外套暗部（非冷蓝/冷绿）
```

**提示词设计**：
```
镜头 A：...暖黄底调 + 浅灰衣服中性调（色调层）。
镜头 B：...暖黄底调 + 深棕外套暗部（色调层）。
```

---

#### 5. 声音桥接

**原理**：上镜头声音延续到下镜头

**示例**：
```
镜头 A（结尾）：老李说话声 + 店内环境音
镜头 B（开始）：老李说话声渐隐 + 店内环境音延续
```

**提示词设计**：
```
镜头 A：音效：老李低沉说话声、店内环境音。
镜头 B：音效：店内环境音延续、小张轻微呼吸声。
```

---

#### 6. 焦点转换

**原理**：焦点从 A 转移到 B，引导注意力

**示例**：
```
镜头 A：前景老李实焦，背景小张虚焦
镜头 B：前景小张实焦，背景老李虚焦
```

**提示词设计**：
```
镜头 A：...浅景深聚焦老李面部，背景虚化...
镜头 B：...浅景深聚焦小张面部，背景虚化...
```

---

#### 7. 运动匹配

**原理**：运动方向/速度一致

**示例**：
```
镜头 A：从左向右跟拍小张跑入店
镜头 B：继续从左向右跟拍小张进店
```

**提示词设计**：
```
镜头 A：...从左向右跟拍，小张跑入画面...
镜头 B：...继续从左向右跟拍，小张进店门...
```

### 光影三层结构

**格式**：
```
光影：[光源层] + [光行为层] + [色调层]
```

**词库**：

#### 光源层
- 夕阳低角度逆光
- 体积光穿透尘雾
- 自发光能量核心
- 霓虹灯漫反射
- 烛光跳动
- 月光冷光

#### 光行为层
- 沙尘漫射柔化轮廓
- 金属表面锈蚀高光
- 雨水折射光斑
- 体积光丁达尔效应
- 镜面反射环境
- 次表面散射

#### 色调层
- 暗金暖底调 + 冰蓝高光冷暖对撞
- 青橙色调 (Teal & Orange)
- 冷蓝底调 + 暖黄强调色
- 单色蓝灰调
- 高饱和暖色盘
- 低饱和冷色盘

**示例**：
```
光影：夕阳逆光暗金色 + 核心冰蓝自发光 + 废墟散射暖光（光源层），
      沙尘漫射柔化轮廓 + 金属表面锈蚀高光 + 体积光穿透尘雾（光行为层），
      暗金暖底调 + 冰蓝高光冷暖对撞（色调层）。
```

## 🎬 导演规则

### 规则 1: 镜头内变化优先于切镜头
**原则**：能用一个镜头的内部变化讲清楚，就不要切镜头。

**技巧**：
- Pull Out/Push In：连续拉远/推进
- Rack Focus：焦点转换
- Tracking：跟随拍摄
- 180 度环绕：围绕角色旋转
- 连续表演：情绪弧线不切
- 动作连续：物理反应链

**目标镜头密度**：
- 真人短剧：15-20 镜头/分钟
- CG 动画：18-22 镜头/分钟
- 电影质感：10-15 镜头/分钟

---

### 规则 2: 对话完整保留优先
**原则**：宁可调整镜头节奏，不删减核心对话。

**对话处理策略**：
| 方法 | 节省时间 | 适用场景 |
|------|----------|----------|
| 语速 1.3 倍 | -30% | 大部分对话 |
| 部分重叠 | -10% | 非关键台词 |
| 画外音处理 | -5% | 情绪性台词 |
| 机械音加速 | -20% | 系统 VO |

---

### 规则 3: CG 动画优势最大化
**必须利用的 CG 优势**：
1. ✅ 连续镜头内变化 (非切镜头)
2. ✅ 完美焦点转换 (Rack Focus)
3. ✅ 复杂特效 (系统/粒子/光芒)
4. ✅ 物理模拟 (头发/衣服/灰尘)
5. ✅ 慢动作 (关键瞬间)

---

### 规则 4: 情绪弧线连续
**原则**：角色的情绪变化应该连续呈现，不要切断。

**情绪弧线示例**：
```
麻木 → 委屈 → 质问 → 愤怒 → 决绝
  ↓       ↓       ↓       ↓       ↓
连续表演，不切镜头
```

---

### 规则 5: 16:9 横屏构图优化
**横屏构图技巧**：
- 三分法：主体置左/右 1/3 处
- 引导线：利用环境线条
- 景深：背景虚化突出主体
- 负空间：留白创造氛围

## 🆘 常见问题

### Q1: 提示词被平台拦截怎么办？
**A**: 检查以下项：
- 是否包含裸英文运镜词（Dolly/Aerial/Crane）→ 改用中文
- 是否包含版权敏感词 → 替换为描述性昵称
- 是否超过 500 字符 → 精简冗余词
- 是否包含写实真人面部 → 改为动画/剪影

### Q2: 生成视频动作不流畅？
**A**: 优化提示词：
- 增加物理逻辑描述（重心转移/流体风阻）
- 明确动作起止状态
- 使用"fluid motion""realistic physics"等词
- 考虑分段生成后拼接

### Q3: 对话与口型不同步？
**A**: 
- 计算对话时长，确保镜头时长足够
- 使用 1.3 倍速提示词
- 选择支持口型同步的模型（Seedance/Kling）
- 添加"precise lip sync"描述

### Q4: 多个角色如何保持一致性？
**A**:
- 为每个角色创建形象描述卡
- 使用@图片引用固定形象
- 每段提示词重复角色关键特征
- 避免角色数量过多（≤3 个主要角色）
