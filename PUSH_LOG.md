## 📋 使用说明

### 推送前检查清单
- [ ] 确认文件内容已审核
- [ ] 确认无敏感信息（密码、密钥、个人隐私）
- [ ] 确认文件命名规范
- [ ] 填写推送记录（下方表格）

### 推送命令
```bash
# 1. 查看变更
git status

# 2. 添加文件
git add <文件名>

# 3. 提交
git commit -m "描述"

# 4. 推送
git push origin main
```

## ⚠️ 禁止推送

以下文件/文件夹**不要**推送到 GitHub：

- [ ] `.env` - 环境变量文件
- [ ] `*.pem`, `*.key` - 私钥文件
- [ ] `config.json` (含敏感信息)
- [ ] `memory/` - 个人记忆文件
- [ ] `__pycache__/` - Python 缓存
- [ ] `node_modules/` - Node 依赖
